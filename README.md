# 不思异：九州

《不思异：九州》是一个以《山海经》与神秘九州远行为核心气质的文字选择、生存资源与横向旅途舞台原型项目。

当前阶段：P0 Web 试玩纵切片已可运行并部署；机器验收通过，等待真人试玩与音频主观听感验收。2026-07-15 收到的新剧本设定已经完成世界观合并评估，尚未改写现有 P0 数据。

线上试玩：[https://webdeploy-green.vercel.app/](https://webdeploy-green.vercel.app/)<br>
试玩密码：`tusun2026`

## 当前核心方向

- 三项核心消耗：车轴、粮草、神志。UI 短名为车、粮、神。
- 底层玩法：节点路线、资源消耗、随机事件、死局防护。
- 表现层：假动态横屏长图 / 横向山海旅途舞台。
- 叙事层：风宪郎陆离与鲛人遗民宛渠奉命采风，记录民疾、韩党罪证与山海异兆。
- 采风层：颂、讽、哀、谶四种写法改变《九州总志》与结局倾向。
- 视频层：短循环动态背景、神志异象、关键地点显影和结局/转场镜头。
- 音频层：低存在感音乐、环境声、行旅声、神志污染音效。
- 文档协作：主线程统筹，长期子线程与 sub-agent 分工研究。
