## 3. 内部快审结论

### 图像快审

建议第一批按三组并行：

- A 组：事件弹窗优先图，方图/4:3，例如故王道分叉、青丘狐灯、黑齿交易、神志告急。
- B 组：横向旅途背景长图，3:1，例如故王道荒原、雷泽赤水、黑齿边市、归墟裂隙。
- C 组：绿幕前景独立素材，例如木车/车队组件、断轴组件、狐灯组件、断碑路标组件。

验收底线：

- 背景长图不能带车队。
- 弹窗事件图优先方图或 4:3。
- 绿幕素材必须纯绿色背景、硬边缘、少羽化、无光晕、无投影、无运动模糊。
- 所有图必须无 UI、无文字、无水印、无现代物品。

### 叙事快审

已形成 12 个画面语义方向，覆盖地点、事件、路遇、危机、结局。

可作为 D 线程转图重点：

- 中原驿外：青铜鸟旗、远处九州裂纹。
- 故王道：断裂王道、无字旧碑、朱砂线。
- 黑齿换物：影子称粮与铜钉。
- 烛龙息图：帛图像胸腔般起伏。
- 鸣蛇入轴：轴心有细蛇般声纹。
- 神志污染：地图、影子、路名逐渐不可靠。

验收底线：

- 不复制现代译文、百科段落、现代图鉴设定。
- 事件文字短，适合弹窗。
- 低神志文本要变疯，但仍保留可判断信息。

### 音频快审

已形成 10 条 Web Demo 音频提示语方向：

- 主循环音乐。
- 旅途环境层。
- 地点抵达。
- 地图打开。
- 迷雾显影。
- 资源上涨。
- 资源下降。
- 车轴告急。
- 粮草告急。
- 神志告急。

验收底线：

- 所有新音频状态默认 `candidate-review / internal demo only`。
- 不做宣传片音乐。
- 不引用现有游戏或影视声音。
- 不要口琴西部主题、枪声、马蹄西部节奏、科幻扫描音、恐怖跳吓。

### 接入规格快审

推荐规格：

| 类型 | 推荐尺寸 | 用途 |
|---|---:|---|
| 方图 | 1536x1536，最低 1024x1024 | 事件弹窗、危机、资源意象 |
| 4:3 | 1600x1200 或 1200x900 | 地点志、事件、路遇叙事图 |
| 16:9 | 1280x720 或 1920x1080 | 当前直连槽位、路线卡、结局图 |
| 3:1 | 2400x800 | 横向旅途舞台背景 |
| 绿幕前景 | 1024x1024 或 2048x1024 | 车队、道具、异象前景，后处理后再接入 |
| 短视频 | 960x540 或 1280x720，3-6 秒 | 少量高价值候选，必须有 poster |

验收底线：

- 静态图优先，视频只做少量候选。
- 短视频失败时必须显示 poster，不阻塞玩法。
- B/C 当前直连规格仍偏 16:9，方图和 4:3 建议先作为候选，待 C 线程确认脚本后再进 ready。

### 风险快审

低风险：

- 自研山海舆图、古道、纸雾、墨线、朱砂、车轴/粮草/神志原创意象。
- 方图/4:3 事件图、横向长背景、无文字无 UI 的原创图像。
- 自研风、纸、墨、铃、车轮、远山环境声。

禁止：

- 使用竞品截图、商店图、录屏、音乐、音效作输入。
- 提示语里写“Out There 风格”“Darkest Dungeon 风格”“FAR 风格”等。
- 星图、星球航线、飞船、科幻 HUD。
- 西部牛仔、枪、酒馆木牌、口琴西部主题。
- DD 式英雄队列、火把压力 UI、技能格、哥特硬黑红阴影。
