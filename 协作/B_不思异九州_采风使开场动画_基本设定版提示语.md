# 《不思异：九州》采风使开场动画 基本设定版提示语

> 用途：游戏开场动画，用动态漫画讲清基本设定，而不是概念预告。  
> 建议总时长：60 秒，10 个镜头，每镜头 6 秒。  
> 画幅：16:9 横屏。  
> 生成方式：先生成统一风格静图，再图生视频。  
> 重要：图片内不要生成文字，所有字幕、旁白、标题后期叠加。

## 0. 这版开场要讲清什么

这不是“世界很神秘”的预告，而是玩家进入游戏前必须知道的设定：

1. 王朝正在整理九州边地，想用诗文、档案和官命重新认识世界。
2. 玩家是被派出的采风使，职责是记录风土、人情、民怨、异象。
3. 每到一地，玩家会听见三种声音：官员之声、民众之声、异象之声。
4. 玩家可以收受贿赂、隐瞒真相、上报灾异、干预事件。
5. 玩家写下的诗文不是装饰，而是治理手段，会改变城市后续命运。
6. 车轴、粮草、神志是旅途压力；诗册、见闻、民声是叙事压力。

一句话定位：

```text
奉王命采风九州，用诗记录世界，也用诗审判世界。
```

## 4. 旁白完整稿

```text
王朝疆域已定，九州风土却仍在册外。

于是朝廷设采风使，奉命西行，采集民歌、风俗与灾异。

你不是将军，也不是商旅。
你带着车队、诗简和王命上路。

车轴会断，粮草会尽，人心也会被荒外一点点磨薄。

每到一地，你会听见三种声音：
官员之声，民众之声，异象之声。

有人求你上报，有人求你隐瞒，也有人先把贿赂递到袖中。

你可以旁观记录，也可以出手干预。
你救下的人，可能会让另一群人失去粮。

你用诗整理一地：
颂其治，讽其弊，哀其民，记其异。

诗送回王朝之后，
一座城会被赈济、封禁、清算，或彻底遗忘。

你写下的不是风景，是判词。
奉命采风，写下即判决。
```

## 6. 生成顺序建议

先生成 3 张校准图：

