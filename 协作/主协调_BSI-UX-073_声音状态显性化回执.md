# 主协调回执：BSI-UX-073 声音状态显性化

日期：2026-06-24

## 1. 本轮目标

入口页此前已有画面素材状态，但音频仍主要藏在试听面板和复制模板里。为了让试玩者和后续线程一眼知道“声音能跑，但仍待复核”，本轮把声音状态显性化。

## 2. 已完成

- `试玩入口.html` 新增“声音状态”区块。
- 标出音乐 `6 首待听`、环境 `1 层临时`、神志 `2 层临时`、短音 `11 临时`。
- 声音状态区块可直达 `试听面板` 和 `F_BSI-F-024` 音频验收清单。
- 入口页与主 Demo 缓存号更新到 `20260624-c159`。
- `qa-check.js` 增加声音状态、缓存号、试听入口和音频清单链接守门。

## 3. 边界

本轮只做入口可见性和 QA 守门，不把临时音乐、临时环境声、临时短音效标为正式资产。音频仍需后续主观试听、授权记录和正式替换。

## 4. 验收口径

通过标准：

- 试玩入口能看到“声音状态”。
- 试玩者能知道哪些声音是待听/临时。
- 入口能进入音乐试听面板。
- QA 能防止声音状态块被后续改版误删。
