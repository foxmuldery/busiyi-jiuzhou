## 2. 现代译文 / 资料站表达红线

| 检查项 | 绿 | 黄 | 红 |

| 地点志 / 抵达文本 | 项目自写，短句，符合九州行旅气质 | 有古籍名词但来源待补 | 直接像百科、资料站、现代译文、公众号段落 |

| 古诗 / API 文本 | 本地兜底或公版短句，内部临时可测 | API 条款/缓存待补 | 现代翻译、现代解释、资料站整理直接进入正文 |
| 低神志文本 | 气质变异但规则仍清楚 | 过于抽象，需 B/C 调整可读性 | 借用竞品旁白腔或现代恐怖作品语气 |

红线句式提醒：

- 不要写百科式“据《山海经》记载，某某具有某某特征”。

- 不要把资料站解释改几个词就放进 Demo。
