# BSI-B-005 路遇事件与可观察资源文本

> 子线程 B：世界观与事件文本  
> 日期：2026-06-14  
> 目标：把“半途停下”写成有山海经九州气味、可复用、可接入 WebDemo 的路遇文本方案。  
> 边界：本文件只做文本与叙事模板设计，不修改 C 线程原型代码。

## 1. 路遇的叙事定位

路遇不是普通随机事件，而是“九州图路线选择的回声”。

玩家在地图上看到一段路的消耗、风险、目的地资源；真正上路后，半途事件负责证明：这些数字不是系统扣费，而是车队正在经过一块不稳定的土地。路遇应让玩家短暂停车、重新确认方向，并承担一个有质感的小选择。

建议路遇保持四段结构：

1. 征兆：远远出现不对劲的声、光、风、脚印、祭痕。
2. 停下：车队必须减速、卸载、祭告、辨路或强行穿过。
3. 抉择：两个选择足够，分别对应资源交换、线索获得或污染规避。
4. 余响：结算不只写“资源变化”，而要写车队带走了什么痕迹。

三层神志文风继续沿用 `clear / uneasy / mad`：

| 层级 | 文风 | 叙事作用 |
|---|---|---|
| `clear` | 现实仍可解释，异象像地理与民俗 | 给玩家可判断的信息。 |
| `uneasy` | 现实细节开始错位，声音、方向、影子不可信 | 提醒风险和代价。 |
| `mad` | 主客倒置，路、车、祠、云像有意志 | 表现低神志，不直接剥夺操作。 |

## 2. 现有三条路遇扩展为模板

### 2.1 轮声入骨：车具征兆模板

适用：`road` 为主，也可用于 `rift` 前的旧路。

核心味道：车不是工具，而是九州路脉的一截骨。车轴响，是路在试探队伍。

通用结构：

```text
title: 轮声入骨 / 轮下空响 / 轴中有路
terrain: road
trigger: 平路、旧车辙、王道残基、断碑附近
resourceBias: 车轴为主，粮草为代价，神志可轻微下滑
sanityShift: clear -> 工具损耗；uneasy -> 声音追随；mad -> 车轮主动说话
choiceA: 停车修轴，车轴恢复，耗粮或耗时
choiceB: 不停赶路，省粮或省时，但神志下滑或埋后续隐患
resultTone: 修过之后也不完全安心，车队带走一段“会响的路”
```

可复用文本骨架：

- `clear`：车队行至半途，前轮忽然空响。路面平整，轴心却像被一枚看不见的石子敲中。
- `uneasy`：空响从前轮传到后轮，又从后轮传回队尾。随从说不是车坏，是路在车里响。
- `mad`：轮在叫你。轴心里有一截小路醒着，正等车队把身子套进去。

### 2.2 路旁无名祠：祭祀问路模板

适用：`market`、`road`，尤其是废关、边市、岔道、有人烟残留处。

核心味道：九州路边的祠不一定供神，也可能供“曾经走错的人”。献祭不是祈福，而是交换方向。

通用结构：

```text
title: 路旁无名祠 / 反香小祠 / 无门之祠
terrain: market | road
trigger: 废关前后、边市路口、旧村旁、路线分叉处
resourceBias: 粮草换神志或线索；绕开则损车轴或加风险
sanityShift: clear -> 民俗遗迹；uneasy -> 祠像刚有人；mad -> 祠追着车队移动
choiceA: 献粟问路，粮草下降，神志恢复或获得路线提示
choiceB: 绕祠而过，保粮但伤车/错过线索/留下不敬余响
resultTone: 祠没有明说答案，只让灰、香、脚印或影子指向某处
```

可复用文本骨架：

- `clear`：路旁出现一座小祠，没有匾额，香灰却是新的。祠后有短路，也有几道不像人的脚印。
- `uneasy`：小祠没有门，里面却像有人刚转过身去。香灰排成一行，正指着你们要去的方向。
- `mad`：祠在路旁，也在路中。你们每走一步，它就近一点。供桌上的空位写着车队的影子。

### 2.3 半途黑云：天象压境模板

适用：`water`、`rift`，也可作为高风险路线通用压力事件。

核心味道：黑云不是天气，而是裂隙、泽雷、羽影、赤水蒸汽合成的“会移动的边界”。

通用结构：

```text
title: 半途黑云 / 地上低云 / 羽影压路
terrain: water | rift
trigger: 泽地、赤水、裂隙前夜、高风险路线
resourceBias: 车轴与粮草互换；神志通常下滑；可埋线索
sanityShift: clear -> 低云暴雨；uneasy -> 云里有羽毛味/路声；mad -> 云在地上，天在车轮旁
choiceA: 卸重避云，保车轴，耗粮草或丢物
choiceB: 循雷/穿云快行，省粮或推进，损车轴与神志
resultTone: 黑云经过后留下灰、湿羽、红泥或反向脚印
```

可复用文本骨架：

- `clear`：远处黑云贴着地面滚来，不像雨，更像一群低飞的影子。车队必须决定避让还是抢行。
- `uneasy`：黑云没有风声，只有湿冷的羽毛味。有人说云里有路，有人说云本身就是路。
- `mad`：云在下面。天在车轮旁边。黑色的东西一张一合，等你们把名字驶进去。

## 3. 地图节点“可观察资源”的叙事表达

地图节点不要只显示“车轴 +10 / 粮草 +8 / 神志 -3”。建议在路线卡或节点详情中增加一行“远望所见”，让玩家先读到资源的世界观形态，再看到数值。

### 3.1 资源表达原则

- 车轴：写成木、铁、石、旧车、门梁、辐条、绳索、可垫路的硬物。
- 粮草：写成粟、芦根、白鱼、黑粟、岸草、可煮之物，也可带禁忌。
- 神志：写成方向感、姓名、碑文、药粉、祭告、旧图、能让人“记得自己是谁”的东西。
- 负面代价不要只写损失，要写“为什么拿它会让人不安”。

### 3.2 当前节点可观察资源短句

| 节点 | 远望所见 | 叙事化资源提示 |
|---|---|---|
| 中原驿 | 驿墙仍有灯，仓门半掩，旧木匣里有铜钉声。 | 可补干粟、借钉修轴；这里的人间秩序还能让队伍稳住。 |
| 故王道 | 石基半埋，深车辙向西，路旁有被撬过的王道石。 | 可取石垫车、绞旧贡绳，但旧路会把人的心也往回拉。 |
| 废关 | 关楼倒塌，门梁露出硬木，铁钉黑得像旧火。 | 可拆门木修车；越靠近旧令，越容易怀疑西行本身。 |
| 雷泽浅畔 | 泥下冒泡，芦根肥白，偶有白鱼翻在浅水里不动。 | 可采芦根、捞白鱼；吃下去的东西带着地底雷声。 |
| 黑齿市 | 日中有棚帐显形，兽骨摊和黑粟袋围成一圈。 | 可用旧铁换粮、用粮换静神药；市价常常不是当场付清。 |
| 赤水外滩 | 红水无波，岸草密，旧骨被摆成路标。 | 可采岸草充饥；骨路能指向裂隙，也会磨损人的判断。 |
| 巫咸断碑 | 黑土中露出半截碑，残笔像方位，也像姓名。 | 可拓碑定方、依碑文正名；看得太久会把自己也读进去。 |
| 九州裂隙 | 舆图像被折错，来路变薄，残图在风里自行翻页。 | 可整束残图稳住神志；这里的补给更像最后一次确认现实。 |

### 3.3 地图卡显示建议

每个可选目的地建议显示三层信息：

```text
远望：关楼倒塌，门梁露出硬木，铁钉黑得像旧火。
可得：车具材料充足；旧令会扰乱人心。
代价：多停留会折损神志，绕行则更耗车轴。
```

这样玩家先理解“为什么这里有资源”，再接受数值结算。

## 4. 路遇事件样例池

以下样例可作为 `routeEvents` 扩展池。资源影响用方向标注，具体数值交给 A/C 线程映射。

### 4.1 轮声入骨

适用 terrain：`road`  
资源影响方向：车轴上升或避免损耗；粮草下降；神志小幅下降或不变。  
神志文风变化：工具故障 -> 声音追随 -> 车轮像活物。

文本：

- `clear`：车队行至半途，前轮忽然发出空响。路面没有石子，轴心却像被什么东西轻轻敲了一下。
- `uneasy`：前轮空响三次，每次都像从队伍后方传来。随从停住脚步，说这声音不像木头。
- `mad`：轮在叫你。轴心里有一小段路醒着，它说再往前一点，就能把车队套进去。

选项：

- 停车紧轴：车轴上升，粮草下降。结果：你们在路心紧过轮楔，车声重新变稳，只是轴缝里夹着一点黑土。
- 不停赶路：粮草小降，神志下降。结果：车队没有停，那声空响很快追到每个人耳后。

### 4.2 路旁无名祠

适用 terrain：`market | road`  
资源影响方向：粮草下降换神志上升或路线线索；绕行则车轴下降。  
神志文风变化：民俗小祠 -> 祠中有人 -> 祠追着队伍。

文本：

- `clear`：路旁出现一座小祠，没有匾额，香灰却是新的。祠后有一条更短的小路，也有几道不像人的脚印。
- `uneasy`：小祠没有门，里面却像有人刚转过身去。香灰边缘排成一行，正指着你们要走的方向。
- `mad`：祠在路旁，也在路中。你们每走一步，它就近一点。祠里供着空位，空位上写着车队的影子。

选项：

- 献粟问路：粮草下降，神志上升，可给路线提示。结果：粟粒落进香灰，灰面显出半截弯路。
- 绕祠而过：车轴下降。结果：车轮压过硬土，像压碎了许多干骨。

### 4.3 半途黑云

适用 terrain：`water | rift`  
资源影响方向：卸重可保车轴但耗粮；抢行伤车轴并降神志。  
神志文风变化：低云压路 -> 云里有路 -> 云吞名字。

文本：

- `clear`：远处黑云贴着地面滚来，不像雨，更像一群低飞的影子。车队必须决定是卸重等它过去，还是抢在云前穿行。
- `uneasy`：黑云没有风声，只有湿冷的羽毛味。有人说云里有路，有人说云本身就是路。
- `mad`：云在下面。天在车轮旁边。黑色的东西一张一合，等你们把名字驶进去。

选项：

- 卸重避云：车轴上升或免损，粮草下降。结果：黑云擦着车辙过去，留下一层像香灰的细黑粉。
- 循雷快行：车轴下降，神志下降。结果：你们追着云下的雷声疾行，雷声里有人替车队数步。

### 4.4 反行车辙

适用 terrain：`road`  
资源影响方向：辨路耗粮但稳定神志；强行沿辙省粮但损车轴。  
神志文风变化：车辙异常 -> 同一队伍走过两次 -> 未来车辙先出现。

文本：

- `clear`：旧道上出现一排新车辙，方向却像从西边回来。泥还湿，路旁没有人。
- `uneasy`：车辙的宽窄与你们的车正好相合。随从蹲下量了量，不肯说第二遍。
- `mad`：你们还没走到那里，车辙已经替你们回来过一次。泥里留着明日的轮印。

选项：

- 停车验土：粮草下降，神志上升。结果：你们等到泥面干裂，终于看出其中一半是水影。
- 沿辙快行：车轴下降，粮草小降。结果：车轮咬进旧痕，行得很快，却像被前方的自己拖着走。

### 4.5 芦下白鱼

适用 terrain：`water`  
资源影响方向：粮草上升但神志下降；放回白鱼可稳神志但耗粮。  
神志文风变化：泽中补给 -> 鱼不该还活着 -> 鱼在梦里游。

文本：

- `clear`：浅泽芦根旁有几尾白鱼搁在泥上，鳞片干净得不像从水里来。
- `uneasy`：白鱼被捞起后才开始挣扎，鱼口一张一合，像在学人说西边。
- `mad`：鱼没有眼。鱼在看你。它们不是被捞上来的，是从你们昨夜的梦里翻出来的。

选项：

- 捞鱼充饥：粮草上升，神志下降。结果：鱼汤没有腥味，只在咽下去时响了一声闷雷。
- 放鱼问水：粮草下降或不变，神志上升/获得浅路提示。结果：白鱼钻回泥下，芦苇倒向一条可走的浅路。

### 4.6 黑齿指价

适用 terrain：`market`  
资源影响方向：粮草与车轴互换；可能小幅神志下降或获得交易线索。  
神志文风变化：异族交易 -> 价格像在数人 -> 市口像活物。

文本：

- `clear`：半途路边忽然多出三张骨摊。黑齿市人指指车轴，又指指粮袋，示意可换。
- `uneasy`：他们笑时没有声音，只有黑齿在日光里一闪。你听不懂价格，却听懂他们在数车上有几个人。
- `mad`：不是摊，是牙。每一张摊都张着口，等你用旧铁、粮、或者明日来付账。

选项：

- 以旧铁换粮：粮草上升，车轴下降。结果：粮袋沉得出奇，袋口的结却像刚被人咬过。
- 以粮换车具：车轴上升，粮草下降，神志小降。结果：新车具合得太准，像本来就从你们车上拆下去过。

### 4.7 羽落无影

适用 terrain：`water | rift`  
资源影响方向：低头疾行耗粮但保神志；接羽可得线索但降神志。  
神志文风变化：羽毛异象 -> 上方有看不见的人 -> 影子离身。

文本：

- `clear`：白羽从天上落下，落在车盖上却没有影子。芦苇间传来很轻的敲击声。
- `uneasy`：羽毛越来越多，却没有鸟。随从说别抬头，因为上方有东西正在等你们认出它。
- `mad`：羽不是羽，是眼。每一片都在往下看，你们的影子被看薄了一层。

选项：

- 低头疾行：粮草下降，神志不变或小升。结果：车队低头穿过羽雨，没人知道头顶那阵沉默有没有离开。
- 以布接羽：神志下降，获得羽民/浅路线索。结果：白羽落进布中，布面显出一段向水上走的痕迹。

### 4.8 赤水回响

适用 terrain：`water | rift`  
资源影响方向：祭告耗粮换神志；循声前进省资源但高神志风险。  
神志文风变化：水声异常 -> 水在学人名 -> 水面回放来路。

文本：

- `clear`：赤水支流横在半途，水不急，却把车轮声一遍遍送回岸上。
- `uneasy`：水声开始像人声，先学随从，再学你。有人听见自己的名字从下游回来。
- `mad`：水没有流。是你们在流。每一声车轮都被赤水收下，换成一个更旧的名字吐回来。

选项：

- 掩骨祭水：粮草下降，神志上升。结果：你们把岸边碎骨埋进土里，水声终于慢了一拍。
- 循声涉过：车轴下降或神志下降，可能解锁近路。结果：车队沿着回声过水，抵岸时少了一道自己的脚印。

### 4.9 巫文倒写

适用 terrain：`road | rift`  
资源影响方向：停车拓印耗粮换神志/线索；毁去刻痕稳定神志但断线索。  
神志文风变化：刻痕像方向 -> 字在补笔 -> 字写进人身上。

文本：

- `clear`：半埋的石片露在路心，刻痕倒着写，像方位，也像一串不完整的人名。
- `uneasy`：你从左读，它从右补。缺的那一笔总在你眨眼时变近。
- `mad`：石头没有字。字在你骨头里。你只是路过时，把它们又读出来了。

选项：

- 停车拓印：粮草下降，获得巫文线索，神志小降或小升。结果：拓片上多出一笔，没人记得是谁拓下的。
- 敲碎石片：车轴小降，神志上升。结果：石片裂开，路心安静了，但西边少了一种解释。

### 4.10 裂隙前梦

适用 terrain：`rift`  
资源影响方向：烧梦图耗粮换神志；按梦行进大幅降神志但可能开近路。  
神志文风变化：疲惫做梦 -> 梦图与现实重合 -> 醒来仍在梦路。

文本：

- `clear`：半途宿营时，队伍中有三人梦见同一张地图。地图上有一条近路，醒来后灰烬里也有那条线。
- `uneasy`：梦图比现实更清楚。现实的路反而像被谁擦淡了，只剩车队脚下这一小段。
- `mad`：你没有醒。车队只是从一个梦驶进另一个梦。裂隙在梦外等，也在梦里等。

选项：

- 烧掉梦图：粮草下降，神志上升。结果：纸灰飞起时，现实的路重新变重。
- 按梦近行：神志大降，可能解锁近路或结局分歧。结果：车队按灰里的线前进，回头时看不见昨夜的火。

## 5. Demo 优先放入的 3 个路遇

推荐第一批放入：

1. 轮声入骨：适合教程后第一条旧王道，能把“车轴资源”变成有体感的东西。
2. 路旁无名祠：适合废关/边市路线，能展示粮草、神志、路线提示之间的交换。
3. 半途黑云：适合雷泽或赤水高风险路线，能做出“横向旅途突然停下”的视觉与音效重点。

这三条刚好覆盖 `road / market / water-rift`，也对应车轴、粮草、神志三种资源压力，适合作为第一版路遇系统的骨架。

## 6. 给 C 线程的数据化备注

当前 `data.js` 已有 `routeEvents` 的字段形态：

```text
title
tag
texts.clear / texts.uneasy / texts.mad
choices[].label
choices[].hint
choices[].effect
choices[].result
```

B 线程建议保持这个结构，不需要新字段也能接入。若后续要扩展事件池，可增加非必需字段：

```text
terrain: string[]
resourceBias: string[]
sanityTone: string
priorityDemo: number
```

第一版不要做复杂随机。建议 C 线程仍按路线指定 `midEvent`，等 A 线程规则确定后，再把同 terrain 的事件放入带权重池。

## 7. 待用户/主协调决策

1. 路遇是否允许“明显正收益”？  
   建议允许，但正收益必须带代价或余响。例如粮草上升，但神志下降；神志上升，但耗粮祭告。

2. 低神志文本是否影响真实选择？  
   建议第一版只影响显示和误读感，不直接隐藏正确选项，避免玩家觉得系统不公平。

3. “神志”最终要偏理智值，还是偏“与九州异象的共鸣值”？  
   目前文本按双义处理：高神志更稳定，低神志更能看见幻路。这个方向很有特色，但需要主玩法决定是否让低神志偶尔解锁路线。

4. 可观察资源要显示到多具体？  
   建议地图卡先显示叙事短句和方向性提示，点击节点或路线后再显示具体数值，避免九州图变成纯数表。
