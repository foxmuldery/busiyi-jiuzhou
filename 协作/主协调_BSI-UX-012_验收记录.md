### 可行路径检查

仅按路线消耗，不使用补给，也能通过以下路径抵达裂隙：

```text
中原驿 -> 故王道 -> 鸟鼠夹道 -> 白羽淖 -> 赤水外滩 -> 巫咸断碑 -> 九州裂隙
```

抵达后资源：

```text

```

这说明当前数值不会把玩家无选择地压死；补给会增加选择空间，而不是成为唯一活路。

## 4. 仍需下一轮处理

1. 当前 `revealPlan` 仍是预制显影，不是完整权重抽卡。
2. 14 个地点可能让小横屏路线卡密度偏高，需要真实试玩看是否裁剪。
3. 音频仍以占位和已有槽位为主，尚未接入 `MAP-FOG-001 / MAP-OPEN-001 / CITY-ARRIVE-001`。
4. 临时服务根目录会导致外部图片/音频相对路径 404；试玩时需用正确目录或统一资源路径。
5. `git diff --check` 仍会被旧文档行尾空格影响，该问题与本轮代码无关。
