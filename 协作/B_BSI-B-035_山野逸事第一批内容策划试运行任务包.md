# BSI-B-035 《山野逸事》第三部第一批志怪内容协作者试运行任务包

日期：2026-06-24

## 1. 本轮目标

把《山野逸事》第三部的旧设定真正接入《不思异：九州》的内容生产管线，而不是停留在“以后可以用”的口头状态。

本轮不追求全量整理，只做一次小闭环：

```text
志怪内容协作者提交 3 到 5 个旧设定、故事桥段或典故来源
→ AI 初读并提出最多 8 个追问
→ 志怪内容协作者自然语言短答
→ AI 生成志怪内容确认版素材包
→ 主线程挑 1 到 2 条转成游戏事件卡
→ 后续择机接入 Web Demo
```

## 2. 志怪内容协作者本轮只需要交什么

请志怪内容协作者任选 3 到 5 个“最容易变成玩家遭遇”的旧设定、故事桥段或典故来源。不要选宏大主线，优先选能在一站或一段路里发生的内容。

这里的志怪内容协作者，是指负责《山野逸事》第三部中部分情节设定、故事桥段和典故来源的人；他不是《不思异：九州》的主策划，也不需要为游戏系统、路线结构或整体世界观负责。

推荐类型：

| 类型 | 适合度 | 例子 |
| --- | --- | --- |
| 奇异地点 | 很高 | 古井、废庙、墟市、渡口、禁林 |
| 民俗禁忌 | 很高 | 不可回头、不可报真名、不可食某物 |
| 怪异交易 | 很高 | 用影子、梦、名字、明日换粮或路引 |
| 异兽路遇 | 高 | 引路、伤车、惑神、偷粮的异兽 |
| 古器残卷 | 高 | 可入行囊或解锁路线的物件 |
| 人物关系 | 中 | 需要能转成地点事件或后续伏笔 |
| 宏大主线 | 暂缓 | 等流程跑通后再拆 |

## 6. 主线程验收门槛

主线程收到《志怪内容确认素材包》后，只挑能进入第一章试玩的小内容。

每条素材进入游戏转译前，必须满足：

| 检查项 | 要求 |
| --- | --- |
| 志怪内容协作者保留态度 | 明确保留或追问后保留 |
| 世界观定性 | 至少知道是真相、谣言、禁忌、骗局、风俗或古国遗痕之一 |
| 玩家欲望 | 玩家知道自己为什么要冒险 |
| 玩家恐惧 | 玩家知道会失去什么 |
| 不可丢意象 | 至少 1 个，最好 2 到 3 个 |
| 低神志方向 | 有正常/低神志差异 |
| 游戏用途 | 能放进地点事件、半途路遇、补给、危机或行囊物 |
| UI 可承载 | 可压成 40 到 70 字正文，选项不超过 4 个 |
| 资源关联 | 能关联车轴、粮草、神志之一 |
| 风险 | 无明显版权、现代表达或项目调性风险 |

## 7. 第一轮建议转译数量

本轮不要把所有确认素材都接进 Web Demo。

建议：

- 从 3 到 5 条旧设定里挑 2 条转成游戏事件卡。
- 其中 1 条做地点事件。
- 其中 1 条做半途路遇或补给事件。
- 最多只接 1 条进 Web Demo 做真实验证。
