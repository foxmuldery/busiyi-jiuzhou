# BSI-B-040 六章路线与 P0 地点映射表

## 结论

六章路线应作为正式版主轴，但当前 P0 Web Demo 不应立刻整表替换。下一轮最适合先把 `central_post / old_king_road / abandoned_pass` 解释为“长安出关 -> 沙墟前哨 -> 沙墟官道”，用最少改动承接第一章《吾身化种》。雷泽、赤水、羽民渡、黑齿市、九州裂隙可暂保留为旧试玩路线，后续按章节需要迁移成苍梧、万丘海、洛尘古国等正式地点。
