## 3. 背景音乐提示语

### MUS-CORE-007 第一章主循环候选

```text
生成一段 2 分 30 秒到 4 分钟的中国古代神秘九州远行游戏主循环音乐，可以无缝循环。画面感是凡人车队穿过古道、远山、水泽、荒关和未知祭所。音乐要古典、优雅、空旷、耐听，有一点不安，但适合长时间阅读文字和做选择。

配器以古琴、箫、埙、低弦、轻微编钟或玉磬、极轻远鼓为主。节奏慢，旋律克制但好听，有一个短小可记忆的主题动机；不要大高潮，结尾能自然回到开头。声音要空，不要满，不能压过文字阅读。

负面要求：不要歌词，不要清晰人声，不要人声唱词，不要现代电子节拍，不要流行歌结构，不要史诗大片感，不要仙侠网游感，不要武侠电视剧主题曲感，不要西部片配乐感，不要恐怖片跳吓，不要模仿任何具体游戏、影视、音乐作品，不要刺耳爆音，不要过度混响。
```

### MUS-EVT-CRISIS-001 危机短音乐

```text
生成一段 12 到 20 秒的中国古代神秘九州游戏危机短音乐，用于禁地、坏结果、资源大幅下降、神志污染突然加重。音乐要低沉、古老、压迫、克制，像玩家误入不可祭祀之地。配器可以使用低弦、埙、远鼓、失谐石磬、很轻的风声纹理。2 秒内建立危险感，但不要跳吓，结尾自然淡出。

负面要求：不要尖叫，不要歌词，不要清晰人声，不要恐怖片大冲击，不要爆炸，不要现代警报，不要战斗音乐，不要史诗大片鼓点，不要强噪音墙，不要模仿任何具体游戏、影视、音乐作品。
```

### AMB-TRAVEL-001 旅途环境层

```text
生成一段 30 到 60 秒的古代神秘荒外旅途环境声，可以无缝循环。包含空旷风声、远处低云流动、极轻木轮和车架摩擦、淡淡布幡抖动、偶尔一两声远处石磬般自然回响。整体安静、空旷、克制，作为背景空气层，不能抢主音乐和文字阅读。

负面要求：不要现代车辆，不要发动机，不要马嘶主导，不要人群说话，不要现代城市环境，不要恐怖尖啸，不要雷暴大声，不要科幻感，不要模仿任何具体游戏、影视、音乐作品。
```

## 4. 短音效提示语

### MAP-OPEN-001 地图打开

```text
生成一个 0.6 到 1.0 秒的古代地图打开音效。像卷轴或帛图轻轻展开，竹轴转动，细微纸帛摩擦，末尾有极轻玉磬或木签点落。清晰、古雅、轻巧，不像现代菜单声。
```

### CITY-ARRIVE-001 地点抵达

```text
生成一个 0.7 到 1.2 秒的古代地点抵达音效。像朱砂印轻落在竹简或帛图上，伴随轻微木盒合拢和短促玉磬尾音。含义是旅途节点被记录，不是奖励音。
```

### RESOURCE-DOWN-001 资源下降

```text
生成一个 0.45 到 0.75 秒的古代资源下降 UI 音效。像干木签滑落、轻微木扣低响、粮袋变轻的一点布料摩擦。克制、略紧张、干净，不要失败 jingle。
```

### WARN-SANITY-001 神志告急

```text
生成一个 1.0 到 1.6 秒的神志告急音效。包含低频不稳定嗡鸣、远处石室回响、非常轻的反向呼吸、不可辨认的低语纹理。阴冷、神秘、低音量，不要跳吓，不要清晰词句。
```

## 5. 版权与近似风险要求

- 全部新生成结果先标 `candidate-review / pending-user-confirmation / internal-web-demo-only`。
- 不写 `cleared`，不放入 `05_可用素材`。
- 不引用具体作曲家、影视、游戏、厂牌、竞品名，不写“像某某”。
- 不要歌词、不要清晰人声、不要可辨认咒语；诗歌只能作为声场意象参考，不能唱出来。
- 生成后记录 `sourceTool / sourceAccount / promptVersion / licenseStatus / reviewStatus / usageScope`。
- 主观复核要听是否像现成游戏、影视主题、仙侠网游、武侠电视剧、史诗预告片或西部片马车配乐；有近似感就弃用。

## 6. WebDemo 试听验收清单

- 音频开关：首次点击后能启声；关闭后所有循环层停止。
- 当前音乐显示：不同地形/地点能显示对应音乐 ID，试听卡能切换并恢复“跟随旅途”。
- 音量：音乐默认 `0.13-0.16` 左右，不压正文阅读；神志层更低。
- 循环：主音乐、神志层、环境层头尾无明显断点或爆音。
- 路线体验：古道、水泽、边市/裂隙的音乐差异可听，但不突兀。
- 神志体验：神志低于 55 有轻污染，低于 25 或危机有重污染；不能刺耳、不能遮字。
- 资源反馈：资源上升/下降、车轴/粮草/神志告急只在关键时刻触发，不能连续骚扰。
- 地图反馈：打开地图、迷雾显影、抵达地点三类声音要互相可分辨。
- 浏览器兼容：素材加载失败时玩法仍可用；页面隐藏后循环音暂停。
- 授权验收：未确认来源和授权前，只能内部试玩，不进入正式可用区。
