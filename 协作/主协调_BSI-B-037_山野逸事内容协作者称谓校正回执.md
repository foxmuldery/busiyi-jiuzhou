# 主协调回执：BSI-B-037 山野逸事内容协作者称谓校正

日期：2026-06-24

## 1. 用户澄清

这位协作者不是《不思异：九州》的主策划，也不是本项目整体方向负责人；他也不是整个《山野逸事》第三部的唯一总负责人。他只负责把《山野逸事》第三部中自己掌握的一部分情节设定、故事桥段和典故来源交接进来。

- `协作/B_BSI-B-030_山野逸事第三部志怪素材接入规范.md`

同时把旧文件名里的“主创评审 / 主创文档 / 主创试运行 / 主创AI问答”改为“志怪内容确认 / 内容协作者文档 / 志怪内容协作者试运行 / 志怪内容协作者AI问答”，并同步了资产清单与引用路径。

## 3. 之后统一口径

部分内容策划 / 志怪内容协作者负责：

- 交接自己负责和掌握的旧资料、旧设定、故事桥段和典故来源。
- 解释哪些内容有价值、哪些意象不能丢。
- 回答 AI 的少量关键追问。
- 确认 AI 是否误读了旧项目内容。

他不负责：

- 《不思异：九州》的总体策划方向。
- 游戏数值、路线池、概率、保底、UI 和代码字段。
- 批量填写游戏素材卡。
- 对 Web Demo 接入结果做最终验收。

## 4. 验收

已检查活动交接入口，不再出现容易误导的“主创评审 / 主创追问 / 主创确认 / 原主策划 / 总编剧”口径；后续如需署名，优先使用“部分内容策划 / 志怪内容协作者 / 旧设定交接人”。
