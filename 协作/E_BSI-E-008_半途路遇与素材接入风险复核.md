## 0. 总判断

本轮“半途路遇 + 横向旅途舞台 + 舞台素材分层接入”可以继续服务第一版试玩。当前核心方向仍然是：

```text
羊皮纸山海舆图
+ 古道车队横向行进
+ 车轴 / 粮草 / 神志三资源
+ 半途停下处理路遇
+ 背景长图、前景车辙、污染层、车队层分离
+ 地图三短音效先占位或静默可玩
```

风险总体为“可控，但素材接入需要黄灯管理”。机制层面借鉴路线选择、资源压力、横向旅途、事件打断都属于常见玩法语法；风险集中在截图级表达、音频/图片授权、以及内部参考语汇外溢。

本轮继续强调：不讨论宣传，不做商店页/PV/对外卖点包装。凡涉及宣传的内容，本文件只列禁区与延后复核要求，不给宣传方案。

## 2. 竞品具体表达禁止清单

### 2.1 Out There 禁区

Out There 的公开页面将其定位为太空探索、roguelike、资源管理和互动小说，并包含程序生成星系、飞船、氧气、外星语言等表达。我们可以借鉴“路线选择与资源压力”的抽象结构，不能接近它的太空表达。

禁止：

- 星图、星球节点、星系网、发光航线、跳跃线。
- 深空背景、星云、扫描圈、雷达波、科幻 HUD。
- 飞船、宇航员、冷冻苏醒、采矿、氧气/船体/燃料三资源对应。
- 外星语言学习、飞船设备槽、星球事件卡的具体结构。
- 把低神志星象层做成常态地图 UI。
- 在 UI、对外文案或素材提示词里写“Out There 风格 / 山海经版 Out There / 同款星图探索”。

安全替代：

- 羊皮纸、帛书、旧舆图、墨线、朱砂、关防印。
- 地名签、祠印、关隘符号、泽痕、车辙。
- 车轴、粮草、神志。
- 雾散、纸背渗出、笔锋描路、朱印落下。
- 荒外游记、路引签、地点志。

### 2.2 Darkest Dungeon 禁区

Darkest Dungeon 的公开页面强调哥特 roguelike、心理压力、队伍英雄、苦难、压力/疾病/黑暗、哥特鸦羽线稿和旁白系统。我们可以借鉴“队伍承压、失败复盘、事件打断”的抽象压力感，不能复制它的角色/UI/叙事组合。

禁止：

- 哥特硬阴影、鸦羽线稿、红黑怪物、扭曲英雄肖像。
- 英雄横排队列、职业头像、站位格、技能格、回合战斗 UI。
- 火把压力条、Affliction/Virtue 式命名、怪癖/病症体系。
- 讽刺、庄严、失败评语式旁白腔。
- 营地、酒馆、修道院式压力治疗结构。
- “中国版 Darkest Dungeon / 山海经 DD”口径。

安全替代：

- 车队资源 HUD，而不是英雄队列。
- 神志阈值影响文本可信度和地图污染，而不是压力病症复制。
- 地点志、残卷、路引、车队记事，而不是旁白裁决。
- 山海异象、巫文、纸雾污染，而不是哥特怪物远征。

### 2.3 FAR / 横向载具旅途禁区

FAR: Lone Sails 的公开页面强调独特载具、保持载具运行、路障、恶劣天气、干涸海床与云风旅程。我们可以借鉴“横向画幅、旅途气氛、假动态、云风经过”的抽象视听语法，不能滑向载具操作游戏。

禁止：

- 大型独特载具剖面、车内横截面、锅炉、引擎、拉杆、操作站。
- 实时驾驶、碰撞、路障解谜、维修灭火、载具升级流程。
- 以“机器与你相依”为核心的情绪表达。
- 同款地平线构图、云风镜头节奏、干涸海床孤旅表达。
- “类 FAR 载具旅途 / FAR-like”口径。

安全替代：

- 车队层只作前景状态和旅途符号，不成为可操作机器。
- 背景/前景视差服务“走在路上”，不做平台碰撞或路障机关。
- 半途停下进入选择事件，核心仍是路线、资源、神志。
- 云雾、风沙、水泽作为山海气候层，不做 FAR 式孤独机器诗。

### 2.4 西部游戏禁区

“横向西行车队感”只能作为内部节奏参考，不能进入西部片/西部游戏具体表达。

禁止：

- 西部马车、牛仔、枪械、赏金、酒馆木牌、荒漠小镇、口琴主题。
- 马蹄、缰绳、枪声、酒馆钢琴等西部声景。
- 黄沙夕阳 + 牛仔剪影 + 木牌 UI 的组合。
- “西部远行换山海皮”口径。

安全替代：

- 古代辇车、牛车/辎重、驿路、旧关、墟市、祭所、泽国。
- 木轴、车轮、粮袋、纸墨、朱印、风雾短音效。
- 山海古道、赤水、雷泽、羽民渡、巫咸断碑等项目自有语汇。

## 3. 低风险可借鉴边界

| 可借鉴对象 | 可借鉴层级 | 九州转译 | 不可越线点 |
|---|---|---|---|
| Out There | 大界面路线选择、资源消耗、未知显现、失败复盘 | 九州舆图上选路；雾散显影；车轴/粮草/神志共同压迫 | 不复制星图、星球、飞船、氧气、科幻 HUD |
| Darkest Dungeon | 队伍承压、精神资源、危险氛围、失败记忆点 | 神志影响文本与地图可信度；荒外游记复盘 | 不复制哥特美术、英雄队列、火把压力、旁白腔 |
| FAR | 横向旅途舞台、视差、云风、行进中停下 | 静态长图 + CSS 假动态；车队前景层半途停下 | 不做载具剖面、维修/驾驶/路障解谜 |
| 西部远行游戏 | 横向队伍在路上、远近景层次、路途节奏 | 古道车队、驿路/废关/泽国景观、路遇打断 | 不出现牛仔、枪、口琴、酒馆、西部马车 |
| 山海经/古籍母题 | 地名、异象、神秘地理、祭祀/巫文意象 | 自研地点志、资源功能、路线代价和神志文本 | 不照搬现代译文、百科、图鉴、现代二创设定 |

低风险的关键是“功能可借，表达自写”。只要玩家截图第一眼读到的是“九州荒外舆图 + 古道车队”，而不是“古风星图 / 哥特远征 / 载具横版 / 西部车队”，试玩风险可控。

## 4. 素材授权红线

### 4.1 图片 / 视频 / 动态层

红线：

- 不使用竞品截图、商店图、logo、UI 图标、角色、宣传图作素材或参考图。
- 不使用现代山海经图鉴、影视/漫画/游戏截图、来源不明网图。
- 不把 `待复核素材` 直接标为可用或正式。
- 背景图不能自带车队；车队必须独立前景层。
- 黑白棋盘假透明图不能当正式透明素材；需重新抠像或导出真透明。
- 低神志星象/血星层不得用于地图常态 UI、路线连线、节点显影。
- AI 提示词不得点名 Out There、Darkest Dungeon、FAR、西部游戏、具体画师、影视、品牌、角色。

最小入库字段：

| 字段 | 要求 |
|---|---|
| 资产 ID | 如 `BG-006`、`CAR-006`、`OV-003` |
| 文件路径 | 写到实际文件，不写模糊目录 |
| 来源方式 | 自制 / AI 生成 / 用户上传 / 第三方库 / 古籍参考 |
| 生成或取得日期 | 至少年月日 |
| 工具/平台 | AI 工具、模型、素材库或制作方式 |
| 输入素材 | 是否用了参考图/视频；参考素材授权状态 |
| 提示词摘要 | 不含竞品、画师、影视、品牌 |
| 授权状态 | cleared / demo-temporary / review-pending / blocked |
| 相似性初筛 | 是否像竞品截图、现代图鉴、具体作者风格 |
| 用途 | 背景、前景、污染层、车队层、宣传候选等 |

### 4.2 音乐 / 音效

红线：

- 不从游戏、影视、短视频平台、音乐平台截取音乐或音效。
- 不使用竞品音乐、主题旋律、环境声组合或声音 logo。
- AI 音乐也必须记录工具、提示词、输入旋律/参考音频和平台条款。
- 用户上传/AI 生成候选在授权未确认前只可标 `review-pending` 或 `demo-temporary`。
- 第三方免费商用音效必须保存下载页、许可证页、作者、日期、用途限制；未留痕前不得接入为正式素材。
- 地图三短音效缺素材时宁可静默可玩，不可临时抓取来源不明音效。

本轮特别结论：

- `MUS-CAND-006_Wheels_On_The_Stone` 可继续作为内部低响度试玩候选，但不能标正式、不能宣传使用。
- `mapOpen / mapReveal / locationArrive` 三槽位当前 missing，不构成授权风险；F 线程补齐时优先程序生成或授权留痕清楚的短音效。
- 地图打开应是纸卷/木轴/纸边摩擦；显影应是墨晕/朱砂/风雾；抵达应是木印/短朱印感。不要雷达、飞船跳跃、马蹄口琴、哥特战鼓、锅炉引擎。

### 4.3 字体 / 品牌 / 第三方内容

字体：

- 已入库字体按 README 记录为 SIL OFL 1.1，内部 Demo 可用。
- 不得单独售卖字体文件。
- 若修改字体文件或打包进商业发布，保留许可证文本并按 OFL 要求复核名称限制。

品牌台标：

- “不思异 / 不思异TV / ABNORMAL”等台标不是通用素材。内部项目使用可暂存，但对外 Demo、官网、视频、商店页必须由主线程确认品牌授权与展示规范。
- 不得引入竞品 logo、Steam/平台 logo、影视 IP 标识、真实人物肖像或名人声音。

诗词/API/文本来源：

- 古典诗词本体多为公共领域，但 API 的数据整理、译注、现代说明、接口服务条款仍需复核。
- 试玩中可用本地兜底短句，但正式发布前应记录来源、接口条款、缓存许可和引用范围。
- 现代译文、百科条目、公众号/资料站段落不能直接进入游戏文案。
