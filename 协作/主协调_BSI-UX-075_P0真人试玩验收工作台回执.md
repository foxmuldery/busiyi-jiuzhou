# 主协调回执：BSI-UX-075 P0 真人试玩验收工作台

日期：2026-06-24

## 1. 本轮目标

P0 仍缺一次真人试玩复盘和听感判断。入口页已经能复制试玩记录、听感模板和本局复盘，但主线程还需要一个固定位置来汇总反馈并形成“是否 P0 通过”的判断。

本轮新增一个独立验收工作台，把人工反馈收成可复制摘要。

## 2. 已完成

- 新增 `GitHub资产区/03_WebDemo/prototype/P0真人试玩验收工作台.html`。
- 试玩入口新增 `P0 试玩验收台` 链接。
- 工作台支持粘贴：
  - `《不思异：九州》5分钟试玩记录`
  - `《不思异：九州》音频听感复核`
  - `《不思异：九州》试玩复盘`
- 工作台按五项 P0 门槛生成结论：
  - 玩家知道下一步
  - 一局能形成复盘
  - 资源压力可读
  - 横屏 UI 不阻塞
  - 声音不打扰
- 新增“反馈证据雷达”，自动提示下一步、复盘、资源、UI、音频五类证据是否已在反馈文本中出现。
- 可复制《P0 真人试玩验收摘要》，并支持本机草稿保存。
- `qa-check.js` 已加入工作台契约守门。

## 3. 边界

工作台不替代人工判断，也不把待复核音乐或素材标成正式资产。它只负责把真人试玩、听感和本局复盘统一收口，方便主线程判断下一步是 P0 通过、需小修还是阻断。

## 4. 验收口径

通过标准：

- 试玩入口能直达 P0 验收工作台。
- 工作台包含五项 P0 门槛。
- 能粘贴三类反馈并生成可复制摘要。
- QA 能防止工作台入口或核心字段被后续改版误删。
