- 发起时间：2026-06-15
- 发起人：主协调线程
- 状态：已下发
- 当前目标：把上一轮“能跑的九州图”推进为“第一章小关卡闭环”。玩家应能看到更多山海地点差异、补给选择差异、路线显影节奏和失败保护。

## 1. 本轮总目标

1. 扩展 Web Demo 的第一章地点池，让路线不只是一条短线，而是有多个资源偏向的选择。
2. 强化补给参与感：每个地点的可补资源不同，玩家能根据车轴、粮草、神志状态做选择。
3. 补上死局保护与可玩性自测：保证迷雾显影不会让玩家无路可走。
4. 延续商业风险红线：羊皮纸山海舆图，不做古风星图，不用竞品宣传口径。

4. 风险交付优先验：是否能给后续资产和宣传提供红线。
