# BSI-B-033 内容策划确认表与 AI 初拆流程

日期：2026-06-24

## 1. 核心原则

《山野逸事》第三部接入《不思异：九州》时，批量整理交给 AI，内容策划 / 设定协作者不做重复拆卡。

这里的“人类内容协作者”特指掌握《山野逸事》第三部部分情节、设定、故事和典故来源的人。后续建议统一称为“《山野逸事》志怪内容协作者”或“原项目部分内容策划 / 志怪设定顾问”。他不应被当成资料员，也不承担《不思异：九州》的主策划职责；他的价值在于解释旧项目内容、指出 AI 不能误读的地方，并确认哪些情节与典故值得迁移。

他参与的是内容迁移，不是游戏策划接管：

- 他不做数值、概率、路线、UI、音频、美术排期。
- 他不需要把旧资料批量拆成标准事件卡。
- 他只对“是否值得保留、如何理解、哪里不能改、低神志如何扭曲”做判断。
- AI 和游戏侧负责把这些判断转成地点、路遇、补给、危机、行囊或结局伏笔。

这位内容策划 / 设定协作者的价值在于判断和补充：

- 什么值得留下。
- 什么必须改写。
- 什么是九州世界里的真相、谣言、禁忌或骗局。
- 玩家为什么会被诱惑。
- 玩家为什么会害怕。
- 哪个意象不能丢。
- 低神志时它如何变得更恐怖。
- 原资料里没有写明、但内容策划心里已经有答案的设定。

## 2. 推荐流程

```text
旧剧集设定 / 古典典故 / 人物异象
→ AI 批量初拆 30 张候选
→ AI 生成内容策划追问清单
→ 内容协作者回答关键问题
→ AI 合并补充为内容策划确认版
→ 游戏策划转成事件卡
→ 主线程审核
→ Web Demo 小批量接入
```

## 3. AI 初拆候选字段

AI 每次提交候选清单，不要求内容协作者逐格填写。候选字段建议如下：

| 字段 | 说明 |
| --- | --- |
| 候选 ID | 例如 `SYYS-001` |
| 原始出处 | 旧剧集文档、典故来源、章节或备注 |
| AI 一句话 | 用一句话说明这个素材是什么 |
| 核心意象 | 3 到 5 个可视化意象 |
| 冲突核 | 人、鬼、神、俗、地形、禁忌之间的冲突 |
| 初步玩法用途 | 地点志 / 路遇 / 补给 / 危机 / 行囊 / 结局伏笔 |
| 初步资源关联 | 车轴 / 粮草 / 神志 / 暂不确定 |
| 风险备注 | 版权、表达、来源不清、过于影视化等 |
| 需要内容策划判断 | AI 提给内容协作者的 1 到 3 个问题 |

## 4. AI 追问清单

AI 读完一批旧资料后，不只输出候选卡，还要输出“需要问内容策划的问题”。

追问必须围绕内容策划判断，而不是让他补表格：

| 问题类型 | AI 应该追问什么 |
| --- | --- |
| 取舍问题 | 这条设定在原剧集中是不是核心？能不能删？ |
| 隐含设定 | 文档没写但你心里已有答案的规则是什么？ |
| 世界观归属 | 这是民俗误解、真实超自然、古国遗痕，还是人为骗局？ |
| 人物关系 | 这里有没有原本设计过但没展开的人、族群、禁忌关系？ |
| 情绪目的 | 你原本希望观众在这里感到害怕、怜悯、敬畏，还是被诱惑？ |
| 游戏化方向 | 如果玩家经过这里，最应该被迫做什么选择？ |
| 禁改边界 | 哪些名字、意象、关系、句子绝对不能改？ |

## 5. 内容策划确认表

内容协作者只在这张表里做判断，不需要重写整张素材卡。

| 候选 ID | 保留状态 | 世界观作用 | 情绪功能 | 玩家欲望 | 玩家恐惧 | 不可丢意象 | 低神志扭曲 | 内容策划批注 |
| --- | --- | --- | --- | --- | --- | --- | --- | --- |
| SYYS-001 | 进入转译 / 追问后再定 / 暂缓 / 丢弃 | 真相 / 谣言 / 禁忌 / 骗局 / 风俗 / 古国遗痕 | 敬畏 / 恐惧 / 怜悯 / 诱惑 / 荒诞 / 成就感 | 玩家想得到什么 | 玩家怕失去什么 | 1 到 3 个 | 看到什么变坏 | 一两句即可 |

## 6. 内容协作者只回答七个问题

每个被保留的候选，内容策划最多回答七个问题：

1. 这个东西为什么值得进入《不思异：九州》？
2. 玩家在这里被诱惑什么？
3. 玩家害怕失去什么？
4. 它在世界观里是真相、谣言、禁忌还是骗局？
5. 哪个意象、动作或句子绝不能删？
6. 它应该让玩家更想走下一站，还是更怕走下一站？
7. 低神志时它怎么变坏？

如果时间紧，只回答第 1、2、5、7 条即可。

## 7. AI 与内容策划的问答规则

AI 每轮最多问 8 个问题，避免把内容协作者拖进长时间访谈。

问题必须满足：

- 具体：指向某条设定、某个人物、某个异象。
- 可回答：不要问“请完善世界观”这种大问题。
- 有用途：回答后能影响保留、改写、事件设计或资源机制。
- 可追问：如果内容协作者回答里出现新概念，AI 可以下一轮再追 2 到 3 个问题。

推荐节奏：

```text
AI 读资料
→ 输出 30 条候选 + 8 个关键追问
→ 内容协作者回答
→ AI 生成“内容策划确认版素材包”
→ 主线程挑 5 到 8 条进入游戏转译
```

## 8. 内容策划批注示例

```text
候选 ID：SYYS-014
保留状态：进入转译
世界观作用：地方禁忌，背后可能是真相
情绪功能：诱惑 + 恐惧
玩家欲望：用少量祭品换粮草
玩家恐惧：发现粮草里混入人的梦
不可丢意象：井口倒映不是人脸、井绳湿而温、粮袋夜里会呼吸
低神志扭曲：玩家读到的粮草数量会变成亲人的名字
内容策划批注：这不是鬼井，是当地人用梦换收成的旧法。玩家第一次可得利，第二次要付更重代价。
```

## 9. 游戏侧接手标准

一条素材进入游戏转译前，只需要满足：

- 内容策划明确保留。
- 有世界观作用。
- 有玩家欲望。
- 有玩家恐惧。
- 有不可丢意象。
- 有低神志扭曲方向。

之后的数值、选项、概率、路线、保底和 UI 文本，都由游戏策划和主线程处理。

## 10. 对内容策划 / 设定协作者的工作说明

请不要把这件事理解成“填表工作”。

AI 会先把资料粗拆出来，并把它看不懂、拿不准、需要你判断的地方列成问题。你要做的是发挥自己对《山野逸事》这部分内容的理解：

- 这东西有没有灵魂。
- 它是不是《不思异：九州》的东西。
- 它该放在玩家旅途的哪个情绪位置。
- 它会让玩家做出什么不舒服但有吸引力的选择。
- 它在低神志状态下如何露出更可怕的一面。
- 原文档没有写出来的潜台词、禁忌、人物关系和世界规则是什么。

只要这些判断清楚，后面的卡片、数值和接入都可以由游戏侧继续完成。
