## 4. 选项设计

选项数量：2 到 4 个。

| 选项 | 玩家动作 | 预期心理 | 车轴 | 粮草 | 神志 | 其他影响 | 结果文本 |
| --- | --- | --- | ---: | ---: | ---: | --- | --- |
| A |  | 稳妥 / 贪婪 / 冒险 / 逃避 | 0 | 0 | 0 |  |  |
| B |  | 稳妥 / 贪婪 / 冒险 / 逃避 | 0 | 0 | 0 |  |  |
| C |  | 稳妥 / 贪婪 / 冒险 / 逃避 | 0 | 0 | 0 |  |  |
| D |  | 稳妥 / 贪婪 / 冒险 / 逃避 | 0 | 0 | 0 |  |  |

填写规则：

- 至少一个选项应有明确好处。
- 至少一个选项应有明确代价。
- 不要所有选项都只扣资源。
- 不要让首章事件单次扣到玩家直接死局。
- 如果有高收益，应有资源、神志、后续标记或未知风险作为代价。
