## 0. 总结判断

用户最新意见“战略选择参考 Out There 的大界面，但用羊皮纸山海地图；城市和路线通过迷雾显现”可以成立，前提是严格拆分：

```text
可以借鉴：大界面路线决策压力、资源取舍、未知探索、到达后节点显现。
不能复制：星图、太空、星球节点、科幻航线、资源图标、UI 排版、文案结构、商店话术。
```

本项目的安全方向应明确为：

```text
羊皮纸山海舆图
+ 九州远行车队
+ 车轴 / 粮草 / 神志
+ 雾散、城显、路显
+ 路引签式路线卡
+ 自研事件与自研文本
```

风险等级：当前方案在“机制抽象”层为低到可控风险；只要 D/C 线程不把地图做成古风星图、不复制节点图标和资源视觉、不在宣传中绑定 Out There，商业风险可控。最高风险点集中在视觉第一印象、路线卡排版相似、宣传话术依附竞品三处。

## 1. 参考 Out There “大战略选择界面”的风险边界

### 1.1 可参考的不是某个画面，而是决策功能

Out There 可作为内部研究对象，用来理解“玩家在一个大界面里比较下一步路线”的压力结构。可参考的是功能层：

- 玩家从一个大地图或战略界面做下一段路的选择。
- 每条路线带来资源消耗、风险和信息收益。
- 远方未知，选择之后才揭开新的区域。
- 短局循环由“选路、承受代价、遭遇事件、抵达、再选路”构成。
- 失败后玩家能复盘“是哪条路、哪个资源、哪个风险导致崩盘”。

这些属于常见玩法结构或抽象设计目标，不能被某一竞品垄断。

### 1.2 不可参考的是可识别表达

如果玩家、媒体或平台看到截图后第一反应是“这是把 Out There 星图换成古风皮肤”，风险会显著升高。必须避开：

- 星图式宇宙背景。
- 星球 / 星系 / 航线节点结构。
- 科幻 HUD、扫描线、雷达圈、太空粒子。
- 与竞品近似的资源图标组合、图形比例和颜色系统。
- 与竞品近似的路线面板、节点布局、事件文案节奏。
- “山海经版 Out There”“中国版 Out There”“精神续作”等外部宣传口径。

### 1.3 内部参考与外部表达分开

内部协作文档可以继续用“Out There 式大界面战略选择感”作为团队沟通捷径，但外部页面、商店页、预告片、开发日志标题、媒体稿不应主动提竞品名。对外只讲自家题材和机制：

```text
一款山海经题材远行生存叙事游戏。玩家管理车轴、粮草与神志，在逐步显影的九州舆图上选择前路。
```

## 2. 可借鉴的抽象机制

| 抽象机制 | 可用等级 | 《不思异：九州》转译口径 | 避险要点 |
|---|---|---|---|
| 大界面路线决策 | 低风险 | 九州图是独立战略界面，玩家在此决定下一段路 | 画面必须是舆图，不是星图或现代节点图 |
| 多资源压力 | 低风险 | 车轴、粮草、神志共同约束选路 | 不沿用燃料 / 氧气 / 船体等太空资源对应关系 |
| 路线风险比较 | 低风险 | 路线卡显示消耗、风险、半途路遇、目的地补给 | 卡片信息顺序可通用，但视觉和文案必须自研 |
| 未知探索 | 低风险 | 远方只露雾中影、残名、方向和地貌 | 不用宇宙未知星域、扫描发现等表达 |
| 到达后节点显现 | 低风险 | 雾散、城显、路显，像旧舆图被墨迹描开 | 不用星球点亮、航线解锁、雷达扫图 |
| 锁定高价值路线 | 可控风险 | 红封、断墨、线索条件、神志阈值 | 不复制竞品的跳跃条件、外星语流程、设备槽 |
| 路线抽卡 / 可控随机 | 低风险 | 从预制地点 / 路线 / 事件池显现下一组路引 | 不把抽卡做成科幻 UI 或手游抽卡爆光 |
| 失败复盘 | 低风险 | 荒外游记、残卷、车队遗物、地点志 | 不复制竞品失败文本、结局结构和宣传卖点 |

## 3. 禁止复制的具体表达

以下内容应作为 D/C/F/B 线程共同禁区，尤其是 D/C 的截图级表达。

| 禁止对象 | 风险等级 | 禁止原因 | 安全替代 |
|---|---|---|---|
| 星图 / 星系 / 宇宙背景 | 禁止 | 竞品识别度最高，容易形成“换皮星图”印象 | 羊皮纸、帛书、旧舆图、墨线山川 |
| 太空主题表达 | 禁止 | 直接继承 Out There 的题材外观 | 九州、荒外、驿路、泽国、关隘、祭所 |
| 圆形星球节点 | 高风险 | 节点形状与空间隐喻过近 | 地名签、朱砂印、关防印、残卷边角、墨点城影 |
| 发光航线 / 跳跃线 | 高风险 | 容易像太空航线 | 朱砂线、淡墨车辙、断墨、红封、旧折痕 |
| 科幻 HUD / 雷达 / 扫描圈 | 高风险 | 与太空探索 UI 语言绑定 | 雾散、水渍、墨晕、笔锋描入、朱印落下 |
| 竞品资源图形 | 禁止 | 图标组合与玩家记忆关联强 | 车轴裂纹、粮袋 / 粟粒、神志残纹，但需自研图标 |
| 竞品 UI 排版 | 高风险 | 大地图、资源、路线、文本的具体比例若接近会加重相似 | 地图主、路引签辅、游记日志自研布局 |
| 竞品节点图标体系 | 高风险 | 即使换主题，符号关系也可能被认为复制 | 每类节点使用山海地理符号：驿、关、泽、墟、碑、隙 |
| 竞品文案结构 | 禁止 | 文案是直接表达，不可仿写 | B 线程自研游记 / 地点志 / 路引语气 |
| 竞品宣传话术 | 禁止 | 依附竞品品牌，且易造成授权误解 | 只讲“远行生存叙事”“逐步显影的九州舆图” |
| 飞船、外星语言、采矿、太空孤独叙事 | 禁止 | 题材组合过近 | 车队、巫文、残卷、荒外异象、王命 / 失路 |
| 以竞品截图作视觉临摹 | 禁止 | 截图级参考最容易导向近似 | 只能用文字化风险边界，不把截图放入素材库 |

## 4. 羊皮纸山海舆图如何降低相似风险

羊皮纸山海舆图不是简单换材质，而是从四个层面改写表达系统：

| 层面 | Out There 高风险方向 | 九州安全方向 | 降险效果 |
|---|---|---|---|
| 空间隐喻 | 太空、星域、航线 | 古地图、山川、泽国、驿路 | 题材识别完全不同 |
| 节点隐喻 | 星球 / 星系点 | 城市、关隘、墟市、祭所、断碑 | 玩家不会把节点读成星球 |
| 路线隐喻 | 跳跃航线 | 车辙、朱砂路线、断墨旧道 | 避开航线视觉记忆 |
| 揭示隐喻 | 扫描、解锁星域 | 雾散、墨迹显影、地名写出 | 交互反馈更像纸面记录 |
| 资源隐喻 | 飞船生存资源 | 车轴、粮草、神志 | 机制虽同为压力，表达完全转译 |
| 文案隐喻 | 太空迷航 / 外星文明 | 荒外游记 / 山海异闻 / 认知污染 | 叙事气质分离 |

关键判断：

```text
如果九州图像一张正在被写出的古舆图，风险下降。
如果九州图像一张古风皮肤的星图，风险上升。
```

因此 D/C 线程应优先确保第一眼气质：

- 暖旧纸底，而不是深色星空底。
- 淡墨山川和折痕，而不是科技网格。
- 朱砂路线和旧车辙，而不是发光连线。
- 地名签和印记，而不是星球圆点。
- 雾中残名和方向影，而不是问号星球。

## 5. D/C 线程 UI 避让清单

### 5.1 D 线程视觉避让

| 模块 | 必须避开 | 推荐表达 |
|---|---|---|
| 地图底 | 深空、星尘、银河、蓝紫科技渐变 | 羊皮纸、帛书、纸纤维、旧折痕、淡墨山川 |
| 节点 | 发光圆点、星球轮廓、行星环、统一圆形节点 | 地名签、朱印、关防印、残卷角、墨点城影 |
| 当前节点 | 科幻定位环、雷达闪烁 | 朱砂大印、车队小旗、淡金纸晕 |
| 可达节点 | 星球高亮、发光外圈 | 金墨 / 朱砂描边，像刚被描出的地名 |
| 雾中节点 | 现代问号、灰色星球、未发现星系 | 方向词、残名、雾中城影、山影墨点 |
| 路线 | 发光航线、弧形跳跃线、能量轨迹 | 朱砂线、淡墨车辙、断墨线、水路纹 |
| 锁定路线 | 科技锁、能量门、设备不足图标 | 红封、断墨、裂纸、封条 |
| 显影动效 | 扫描圈、雷达波、星域点亮 | 雾散、墨晕、笔锋划过、朱印落下 |
| 路线卡 | 科技面板、飞船控制台、同款资源栏 | 路引签、驿牒、残卷批注 |
| 资源图标 | 燃料罐、氧气、船体、太空仪表 | 车轴裂纹、粮袋、神志残纹，全部自研 |

### 5.2 C 线程实现避让

| 实现点 | 风险动作 | 推荐动作 |
|---|---|---|
| `renderMap()` | 无条件渲染全量路线和全量节点，让玩家提前读完整地图 | 只渲染当前、已见、可达、锁定、雾中；未显现直接隐藏 |
| 可见性状态 | 用“未知星域 / 星图 / jump”等公开或 UI 字段命名 | 用 `fogged / hinted / revealed / traveled / reachable` 等中性语义 |
| 路线显现 | 到达后一次性打开大半张地图 | 每次只显现当前城市一跳路线和少量雾中远影 |
| 雾中节点 | 显示完整地名、补给、路线关系 | 只显示方向、残名或影子，不进地点志 |
| 路线卡 | 列出全图路线，或者把锁定路线刷满 | 只列当前地点已显现路线；锁定高收益最多少量出现 |
| 抽卡表现 | 做成大爆光、稀有度、科幻抽卡 | 路引签淡入、墨线描出、雾边显影 |
| 资源图标 | 直接找竞品相似图标或沿用太空资源隐喻 | 自研车轴 / 粮草 / 神志符号，保持古舆图气质 |
| 测试验收 | 只验功能，不看第一眼截图 | 必须看 1366x768、1280x720、小横屏截图是否像舆图 |
| 文案字段 | 复用竞品式事件句式、短描述结构 | 用 B 线程自研的地点志、游记、路引语气 |

### 5.3 D/C 共同验收问题

1. 第一眼是否像羊皮纸山海舆图，而不是星图？
2. 当前城市是否像车队所在的古地图点位，而不是玩家飞船所在星球？
3. 可走路线是否像旧道 / 车辙 / 朱砂批注，而不是航线？
4. 雾散是否像纸面显影，而不是雷达扫描？
5. 路线卡是否像路引签，而不是太空生存面板？
6. 玩家是否只能看到当前已知区域，而不是提前看到完整路线网？
7. 截图拿掉标题后，是否仍然能看出是九州远行，不会被误认成古风太空路线图？

## 6. 宣传和商店页禁用语 / 推荐语

### 6.1 禁用语

| 禁用语 | 风险等级 | 原因 |
|---|---|---|
| 山海经版 Out There | 禁止 | 直接依附竞品品牌 |
| 中国版 Out There | 禁止 | 强绑定竞品，易造成换皮印象 |
| Out There 精神续作 | 禁止 | 可能暗示授权或继承关系 |
| Out There-like / 类 Out There | 高风险 | 不必要地提高比较和投诉风险 |
| 复刻 Out There 的路线选择 | 禁止 | 直接承认复制具体设计 |
| 太空探索换成山海经 | 禁止 | 明确指向换皮 |
| 同款资源管理体验 | 高风险 | 暗示机制与表达都相似 |
| 星图式山海探索 | 高风险 | 会把视觉带回竞品识别区 |
| 克苏鲁山海经 | 高风险 | 容易造成审美依附和标签错配 |
| 最真实还原山海经 | 高风险 | 绝对化宣传，且与自研转译冲突 |
| 完整九州开放世界 | 高风险 | Demo 阶段承诺过度 |

### 6.2 推荐语

| 场景 | 推荐表达 |
|---|---|
| 一句话介绍 | 一款山海经题材远行生存叙事游戏。 |
| 核心玩法 | 管理车轴、粮草与神志，在逐步显影的九州舆图上选择前路。 |
| 地图卖点 | 城市和路线不会一次性摊开，车队每抵达一处，雾才散开，新的路才被描出。 |
| 风险压力 | 每条路都有代价：断轴、缺粮、失神，或者看见不该看见的东西。 |
| 叙事气质 | 你不是在收集怪物图鉴，而是在写一卷可能没人能读懂的荒外游记。 |
| Demo 范围 | 当前 Demo 展示第一张地图的核心循环：选路、消耗、路遇、显影与失败复盘。 |
| 开发中说明 | 地图、美术、音频与数值均为开发中版本，正式版会继续复核素材授权和内容表达。 |

### 6.3 外部页面推荐短描述

```text
《不思异：九州》是一款山海经题材远行生存叙事游戏。你带着车队穿过逐步显影的九州舆图，管理车轴、粮草与神志，在每一段路的代价和未知之间做选择。
```

更口语的社区版本：

```text
这不是打怪升级的山海经游戏。你带着一支车队往荒外走，车轴会断，粮草会空，神志低了以后，连地图和文字都未必可信。
```

## 7. 风险等级矩阵

| 风险项 | 当前等级 | 触发方式 | 避险措施 | 责任线程 |
|---|---|---|---|---|
| 竞品依附宣传 | 高 | 商店页、预告片、媒体稿出现“山海经版 Out There”等说法 | 对外只讲自家题材和机制，不主动提竞品名 | E / 主线程 |

| UI 排版近似 | 高 | 地图、资源、路线卡比例和信息层级接近竞品截图 | 地图主、路引签辅、游记日志自研；不临摹截图 | D |
| 资源图标近似 | 高 | 资源图标、颜色、排列让人联想到竞品太空生存资源 | 自研车轴 / 粮草 / 神志图标，避免太空仪表感 | D / C |
| 路线节点图标近似 | 高 | 使用圆形星球、发光节点、航线点亮 | 地名签、朱印、关防印、断墨、车辙 | D |
| 文案结构近似 | 高 | 事件、路线、失败文本模仿竞品节奏和句式 | B 线程自研地点志、路引和游记语气 | B / E |

| 现代资料版权 | 高 | 使用现代译文、百科、图鉴设定进入正文或商店页 | 只用原典名词和自研转译，建立资料台账 | B / E |
| AI / 临时素材披露 | 高 | 临时素材被当正式素材用于商店页或宣传图 | 授权台账，公开测试说明区分临时与正式 | D / F / E |

## 8. 给主线程的执行建议

1. 批准 D/C 继续按“羊皮纸山海舆图 + 迷雾显影”推进，不需要因 Out There 参考本身停工。
