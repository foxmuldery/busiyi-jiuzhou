# B_BSI-B-011 第一章地点志与路遇文字试玩压缩版

> 子线程：B，山海经世界观与叙事主创  
> 任务：BSI-B-011  
> 范围：只写协作文档，不修改 `data.js` 或其它代码  
> 用途：给 C 线程直接替换第一章 `locations[].detail` 与 `routeEvents[].texts`

## 0. 试玩版文本口径

- 地点志只保留 2-3 句：先给眼前所见，再给山海经异象，必要时点一下资源取舍。
- 半途路遇只保留“正常 / 低神志”两档：正常说清发生了什么，低神志让错觉压上来。
- 不写百科条目，不解释出处，不做宣传语。
- 小横屏优先短句。少用长逗号，少用抽象词，多用车辙、灯、羽、骨、碑、纸背。

## 3. 字段建议

```js
locations: {
  thunder_marsh: {
    detail: "雷泽水浅，雷声却从泥下滚过。芦根能吃，白羽和断绳提醒你别在水声里待太久。"
  }
}
```

半途路遇：

```js
routeEvents: {
  wheel_omen: {
    texts: {
      clear: "半途前轮空响，路面却没有石子。轴心像被人从里面轻敲了一下。",
      uneasy: "空响从队尾追上来。轴心里有一小段路醒着。"
    }
  }
}
```
