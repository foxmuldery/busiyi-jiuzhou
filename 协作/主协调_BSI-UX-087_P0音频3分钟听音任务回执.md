# 主协调 BSI-UX-087 P0 音频 3 分钟听音任务回执

日期：2026-06-24

## 1. 本轮目标

降低 P0 音频听感人工复核的执行阻力：不是替代人工听感，而是让听音人能一键收到清晰任务，主线程能更快回收可用摘要。

## 2. 已完成

- `P0音频听感验收工作台.html` 顶部新增 `复制听音任务`。
- 听音任务内容包含：
  - 音频验收台入口。
  - 游戏内试听入口。
  - “只判断听感，不判断授权”的边界提醒。
  - 先听 6 首音乐候选，再听环境、神志、告急、短音、地图层。
  - 保留 / 待改 / 弃用的判断标准。
  - 最后回传 `P0 音频听感验收摘要`。
- 复制失败时，任务书会自动写入 `可复制验收摘要` 文本框，供手动复制。
- `p0-readiness-check.js` 已把声音待人工项的下一步改成：复制 3 分钟听音任务，并回收音频验收摘要。
- `qa-check.js` 已加入守门项，防止音频任务入口被误删。
