# C_BSI-C-018 左下角文本入口简化回执

## 触发

用户反馈：左下角版面设计不合理，文字区域太多；希望用小图标或弹出窗口让整体画面更简洁。

## 已调整

1. 左下角“地点志 / 当前遭遇”不再常驻长正文。
2. 默认状态改为两个小入口：
   - `志`：打开当前地点志。
   - `遇`：打开当前遭遇正文。
3. 完整地点志、诗句、地点图、遭遇正文统一进入居中弹窗。
4. 弹窗支持关闭按钮、遮罩关闭和 `Esc` 关闭。
5. 左下角宽度明显缩小，把空间还给补给与剧情选择区。

## 验证结果

- `app.js` 语法检查通过。
- `data.js` 语法检查通过。
- `git diff --check` 通过。
- 932 × 430 横屏：
  - 左下角入口宽约 151px。
  - 补给区宽约 273px。
  - 选择区宽约 471px。
  - 页面无整体溢出。
- 844 × 390 横屏：
  - 左下角入口宽约 136px。
  - 补给区宽约 246px。
  - 选择区宽约 424px。
  - 页面无整体溢出。
- 点击 `志` 可打开地点志弹窗。
- 点击 `遇` 可打开当前遭遇弹窗。
