# B_BSI-B-025：山海经亮点文本扩展包

交付日期：2026-06-20  
交付线程：山海经世界观与叙事主创  
主线程用途：作为下一轮地点志、遭遇、低神志文本和图片需求的内容候选。

## 1. 现有风格诊断

现有第一章文本已经形成稳定 Demo 口径：短句、物象先行、玩法信息不丢。最有效的词根是“车辙、灯、羽、骨、碑、纸背、名字、影子、价码”，每段通常承担一个资源判断：车轴、粮草或神志。

低神志文本最好继续保持“世界错位，但玩家还能看出风险方向”，不要写成纯氛围谜语。

当前可加强的“山海经亮点”不是堆更多异兽名，而是让城镇、祭所、墟市、禁地各有独特规则：城镇问名，祭所换路，墟市看影出价，禁地让低神志看见不该看见的路。

## 3. 接入建议

先接 P0 三条：`狐册问名`、`灰后短路`、`影摊补价`。它们覆盖城镇、祭所、墟市，且不需要新增复杂路线，只要作为地点事件或随机路遇即可。

第二轮接 P1：`羽账轻渡`、`白鱼闭眼`、`赤盐掌纹`，强化水域和禁地段的山海经辨识度。

P2 两条适合裂隙前或低神志专属，不建议第一轮全开，避免玩家前 5 分钟信息过密。

## 4. 主线程采纳建议

下一轮最小可验证内容建议只接入 P0 三条中的 1 到 2 条，先观察文本密度、资源压力和图片需求是否显著提升“山海经亮点”。其余作为内容池，不急于一次性塞进首章。
