# E_BSI-E-010 下一轮物料生成商业风险边界

> 子线程：E 商业合规与竞品研究顾问  
> 日期：2026-06-19  
> 对应任务：BSI-UX-020 全线程物料提示语开工  
> 结论性质：产品侧可执行风险边界，不构成法律意见  
> 当前原则：本轮 AI 生成物料只进入内部 Demo 候选，不得标为正式可商用

## 0. 总结论

本轮可以继续大量生成图像、音频、视频候选，但必须遵守三个底线：

1. 不点名竞品、影视、画师、音乐人、厂牌、品牌、角色。
2. 不生成“像 Out There / 西部横屏旅途游戏 / Darkest Dungeon”的具体表达。
3. 所有 AI 候选先标 `Demo 临时` 或 `待复核`，不得写成 `cleared / 正式可商用 / 可发布素材`。

风险分级：

| 等级 | 说明 | 本轮处理 |
|---|---|---|
| 低风险 | 自研世界观、自研文本、自制或 AI 生成但不指向具体作品的占位素材 | 可生成，进入内部 Demo 候选 |
| 可控风险 | 使用公共神话母题、山海经名词、OFL 字体、程序合成音效、AI 临时素材 | 可用，但记录来源和提示语 |
| 高风险 | 提示语接近竞品美术/镜头/音乐/文案，或素材来源链不完整 | 只可待复核，不直接接入 |
| 禁止 | 复制竞品表达、现代译文/百科/插画、影视/游戏截图/音乐、明确非商用素材 | 不生成、不接入、不入库 |

## 1. 图像生成风险提示

### 1.1 低风险

- 山海经式古道、废关、赤水、雷泽、墟市、断碑、荒泽、旧驿等原创场景。
- 方图 / 4:3 事件图：只表现事件核心物件、地貌、异象，不带 UI 和文字。
- 横向背景长图：只表现环境，不带车队，不带角色主体，不带可识别现代物品。
- 绿幕素材：纯绿色背景、硬边缘、少羽化、无光晕、无投影、无运动模糊。
- 神志污染层：裂纸、墨污、错位山影、第二月亮、倒流水纹、重复车辙等抽象异象。

### 1.2 可控风险

- 山海经异兽/异族母题：可以用名称和意象，但画面必须原创，不照现代图鉴。
- 古籍感：可以用帛书、竹简、朱砂、墨线、旧纸，不使用真实出版物扫描。
- 星象/天象：只作为低神志异象或远景，不作为常态战略地图路线/节点。
- 车队素材：必须独立前景层；背景长图不能自带车队。

### 1.3 高风险

- “古风星图”“发光路线网络”“圆形节点漂浮宇宙感”容易贴近 Out There。
- “横向荒野载具孤旅”“巨大机器剖面”“车内操作站”容易贴近 FAR 类载具旅途。
- “哥特红黑英雄远征”“横排队伍肖像”“火把/压力条”容易贴近 Darkest Dungeon。
- “牛仔、酒馆木牌、枪械、口琴、赏金、黄沙小镇”容易滑向西部旅行游戏。

### 1.4 禁止

- 竞品截图、商店图、角色、UI、地图、图标、画面构图。
- 现代《山海经》图鉴、现代译本插图、百科图片、公众号配图。
- 影视、游戏、漫画、真实品牌、名人肖像、已有 IP 角色。
- 提示语写“像某某游戏 / 某某画师风格 / 某电影镜头 / 某厂牌概念图”。

## 2. 音频生成风险提示

### 2.1 低风险

- 程序合成短音效：点击、资源涨跌、告急、地图显影、抵达落点。
- 自研抽象声景：木轴、粮袋、旧纸、朱印、风雾、水泽、远雷、低鼓、模糊低语。
- 无歌词、无可识别旋律、无具体语言的人声质感。

### 2.2 可控风险

- AI 生成音乐/环境声：可作为内部 Demo 候选，但要记录工具、日期、提示语。
- 神志污染层：可用倒放感、错拍、低频、远声，但不能像具体恐怖片或游戏音效。
- 民族器乐质感：可写“埙、箫、鼓、弦的质感”，不要指定具体曲目、乐团、演奏家。

### 2.3 高风险

- “像 Sunless Sea / DREDGE / Darkest Dungeon / FAR 的音乐”。
- “西部口琴、马蹄、酒馆钢琴、枪声”。
- “哥特战鼓、修道院合唱、火把压力氛围”。
- “科幻扫描、雷达、飞船跳跃、宇宙低频引擎”。

### 2.4 禁止

- 从影视、游戏、音乐平台、短视频平台截取或采样。
- 提示语点名具体作曲家、歌手、厂牌、影视、游戏、角色。
- 使用清晰歌词、人声唱词、可识别旋律或明显临摹现成配乐。
- 把用户上传/AI 生成音乐候选标为正式 cleared。

## 3. 视频生成风险提示

### 3.1 低风险

- 4-8 秒低运动背景循环：云雾、风沙、水纹、天色微变。
- 0.8-1.5 秒转场：墨迹扫过、朱印落下、雾散显影、旧纸翻折。
- 低神志叠层：错位山影、重复车辙、倒流水纹、第二月亮淡入。

### 3.2 可控风险

- 方图/4:3 事件短动画：只做轻微风、雾、水、纸、灰、影变化。
- 背景循环可以用于舞台容器，但不能进入 UI 操作层。
- 所有视频必须有静态图 fallback。

### 3.3 高风险

- 横向车队长镜头若过度像西部车队或 FAR 式孤旅。
- 载具剖面、机械操作、维修、障碍通过等视频。
- 星图显影、发光航线连接、宇宙式镜头推进。
- 哥特队伍行进、红黑怪物、英雄肖像变形。

### 3.4 禁止

- 影视片段、游戏录像、预告片画面、竞品镜头。
- 用竞品视频作为参考视频或首帧。
- 背景长图/视频自带车队，导致车队不能作为独立前景层。
- 输出带水印、文字、logo、UI、真实品牌标识的视频。

## 4. 竞品借鉴边界

| 参考方向 | 可借鉴 | 不可复制 |
|---|---|---|
| Out There | 大界面路线选择、资源压力、未知显现、失败复盘 | 星图、星球节点、发光航线、飞船、氧气、外星语言、科幻 HUD |
| 西部横屏旅途游戏 | 横向旅途感、远近景层次、队伍在路上的节奏 | 牛仔、枪、赏金、酒馆、口琴、马蹄、西部马车、黄沙木牌 UI |
| Darkest Dungeon 式行进 | 队伍承压、危险气氛、失败记忆点 | 哥特红黑线稿、英雄横排、职业头像、火把压力、Affliction/Virtue、旁白裁决腔 |
| FAR 类载具旅途 | 横向画幅、视差、云风、途中停下 | 载具剖面、锅炉/引擎/拉杆、实时驾驶、路障解谜、维修灭火 |

安全转译方向：

```text
羊皮纸九州舆图
+ 古道车队独立前景层
+ 山海异象与地貌
+ 车轴 / 粮草 / 神志
+ 半途路遇弹窗
+ 低神志纸墨污染
```

## 5. AI 生成素材来源记录字段

每个生成素材至少记录：

| 字段 | 说明 |
|---|---|
| 资产 ID | 例如 `EVT-RED-001`、`BG-ROAD-020`、`SFX-MAP-001` |
| 类型 | 图像 / 视频 / 音乐 / 音效 / 污染叠层 / 绿幕前景 |
| 用途 | 地点图、事件图、路遇图、背景长图、资源音效等 |
| 生成工具 | 工具名、模型名或平台名 |
| 生成日期 | 年月日 |
| 正向提示语 | 完整保留 |
| 负面提示语 | 完整保留 |
| 输入素材 | 是否使用参考图、首帧、音频、视频；来源是什么 |
| 输出文件路径 | 原始文件和处理后文件分开记录 |
| 处理记录 | 裁切、压缩、抠图、转码、降响度、循环点等 |
| 相似性初筛 | 是否像竞品、影视、画师、音乐人、品牌、现代图鉴 |
| 当前状态 | `Demo 临时` / `待复核` / `禁止使用` / `可用候选` |
| 复核人和日期 | 便于追溯 |

状态规则：

- `Demo 临时`：可内部测试，不可标正式。
- `待复核`：来源/授权/相似性未清楚，不能进入可用素材区。
- `禁止使用`：命中竞品、现代译文/图鉴、影视/游戏、非商用等红线。
- `可用候选`：只表示候选质量和来源较清楚，仍需主线程验收后再进入可用区。

## 6. 提示语表达禁用清单

以下写法应直接规避：

| 禁用表达 | 原因 | 替代写法 |
|---|---|---|
| `Out There 风格`、`山海经版 Out There`、`科幻星图改古风` | 过度贴近竞品定位与 UI | `羊皮纸九州舆图，朱砂路线，雾中地名显影` |
| `Darkest Dungeon 风格`、`哥特远征队伍`、`火把压力条` | 贴近具体美术和系统表达 | `车队承压，纸墨污染，神志低落的行旅记录` |
| `FAR: Lone Sails 风格`、`载具剖面横版旅行` | 贴近载具操作与镜头 | `横向古道旅途舞台，远景山影，独立前景车队层` |
| `西部马车旅行`、`牛仔荒漠`、`口琴马蹄` | 滑向西部题材 | `古代辎重车队，旧驿古道，风沙与车轴声` |
| `像某画师`、`某电影镜头`、`某游戏截图质感` | 指向具体作者/作品 | `低饱和旧纸质感，粗粝墨线，原创山海异象` |
| `山海经现代图鉴风`、`百科插图式异兽` | 贴近现代资料整理和图鉴表达 | `从古籍名词出发原创化，异兽只作远影或局部征兆` |
| `透明背景棋盘格` | 网页生成易出假透明 | `纯绿色背景，硬边缘，少羽化，无阴影` |
| `带 UI 字体/按钮/文字说明` | 后续接入难，易混入文字错误 | `无文字、无 UI、无水印、无 logo` |

## 7. 推荐通用负面提示语

图像 / 视频通用：

```text
不要文字，不要水印，不要 logo，不要 UI，不要现代物品，不要枪械，不要牛仔，不要西部酒馆，不要科幻星图，不要星球节点，不要发光航线，不要飞船，不要宇航员，不要哥特英雄队列，不要火把压力条，不要载具剖面，不要机械操作室，不要影视截图感，不要游戏截图感，不要现代山海经图鉴风，不要模仿具体画师或具体游戏。
```

绿幕前景素材：

```text
纯绿色背景，主体完整，硬边缘，少羽化，无光晕，无投影，无运动模糊，无文字，无水印，无 UI，不要透明棋盘格，不要复杂背景，不要把主体裁断。
```

音频通用：

```text
不要歌词，不要清晰人声唱词，不要现代流行歌结构，不要西部口琴，不要马蹄，不要枪声，不要科幻扫描音，不要飞船引擎，不要哥特战鼓，不要恐怖片跳吓，不要模仿具体游戏、影视、作曲家、歌手、厂牌或现成旋律。
```
