## 3. 技术实现

修改文件：

- [app.js](/Users/yuanzhe/Documents/game/GitHub资产区/03_WebDemo/prototype/app.js)
- [data.js](/Users/yuanzhe/Documents/game/GitHub资产区/03_WebDemo/prototype/data.js)

实现点：

- 新增 `randomRouteEvents` 数据池，不覆盖原有固定 `routeEvents`。
- 新增 `allRouteEvents`，让固定路遇和随机路遇共用渲染、结算和校验流程。
- 路遇结算 key 从“路线 + 固定事件”改为“路线 + 当前事件”，避免随机事件和固定事件互相覆盖。
- 新增稳定随机抽取：同一局同一个随机事件不重复刷，抽取由 `runId + routeId + day + traveledRoutes` 稳定决定。
- 抽取会参考路线地形、资源危险状态、`badLuckMeter`、事件池标签和 `rescueCandidate`。
- 资源危险或坏运气偏高时，提高补给/休整类随机事件权重，降低纯惩罚体验。
- 地图路线卡保留固定路遇提示，同时提示“或遇随机异象”。
- 行囊九宫格新增 8 个随机事件 flag 的短标签。
- `poetryContent.sceneVerses.routeEvents` 增补 8 条随机事件古辞。
