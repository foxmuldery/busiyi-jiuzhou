## 1. 第一张测试地图定位

第一张测试地图建议定位为“中原边境到九州异域边缘”。

玩家不应一开始就抵达昆仑、归墟、幽都等终局神域，而是从仍可辨认的人间驿路出发，逐步进入地理、语言、时间和身体感知都开始失准的区域。第一张地图的叙事作用是让玩家明白：路不是背景，地点会消耗队伍，也会改变队伍理解世界的方式。

核心气质：

- 起点仍有人间秩序，玩家可以判断。
- 低风险区让玩家熟悉车轴、粮草、神志三种压力。
- 中风险区开始出现异族、巫文、误读和路线诱惑。
- 高风险区把资源压力和认知污染绑在一起。
- 终局/禁地不一定是胜利点，而是第一张地图的边界、试炼和失败叙事出口。

## 4. 3 个神志文本变化示例

这些示例用于定义文本方向，不作为最终游戏正文。

### 示例 1：赤水外滩

高神志：

> 前方有赤色浅水，水面少波。岸上多旧骨，草从骨缝间生出。随从建议日落前离开。

中神志：

> 赤水不流，却有潮声从车底传来。岸边旧骨像被人重新摆过，草尖一齐朝向队伍。

低神志：

> 水没有来。骨先来了。草在数人，数到你时停住。有人说夜里才浅，夜里才浅。

### 示例 2：黑齿市

高神志：

> 午时见市，货摊沿土墙排开。市人齿黑，言语难辨，但手势尚可推测。

中神志：

> 他们笑时不露声，只露出黑色的齿。你听不懂价格，却听懂了他们在数车上有几个人。

低神志：

> 不是市。是口。摊位一张一合。黑齿在买你们的明日，找零是一袋昨日的粮。

### 示例 3：第二废关

高神志：

> 前方又见废关，形制与来处相似。门额残字倒置，或是旧关重修，或是队伍误入环路。

中神志：

> 废关像在等你认出它。墙缝里的旧灰还温着，仿佛你们刚从另一面穿过。

低神志：

> 关门没有倒。倒的是路。你从门里出来，又从门外进去。旧令写着：不得归。

## 5. 异族语言与巫文机制第一版方向

第一版网页版原型中，异族语言和巫文建议先做成“理解度状态 + 选择风险”的轻机制，不做复杂破译系统。

### 5.1 异族语言

用途：

- 用于异族地点和交易事件。
- 让玩家在信息不完整时做选择。
- 提供重玩价值：同一事件在理解度提升后显露新选项。

第一版表现方式：

- 初见时只显示少量可辨词，例如“粮”“路”“夜”“不可”。
- 理解度提升后，选项提示从模糊变得稍清楚。
- 低神志时可能出现伪翻译，玩家以为自己懂了，其实是污染文本。

建议状态：

- 未识：只能通过手势、物品、行为判断。
- 略识：能辨认关键词，但无法理解语气和禁忌。
- 通意：可看见更多交易条件、路线警告或隐藏代价。

### 5.2 巫文

用途：

- 用于碑、骨、鼓、界标、梦中地图。
- 连接隐藏路线、禁忌提示和神志污染。
- 让“读懂”成为高风险高收益行为。

第一版表现方式：

- 巫文不是谜题文字，而是会反向影响读者的文本。
- 玩家可选择读、拓、毁、祭、绕开。
- 读懂可能获得路线或事件真相，但损神志。
- 神志低时会读出不可靠内容，甚至把普通地名读成自己的名字。

建议状态：

- 不可辨：只知道此处有刻痕或符号。
- 可摹写：能带走拓片，稍后再解。
- 可误读：出现提示，但不保证正确。
- 可通神：获得强线索，同时触发高神志代价或特殊结局线索。

## 6. 横向山海旅途舞台适配

第一版地图若采用“假动态横屏长图/横向山海旅途舞台”，地点不应只是孤立卡片，而应被组织成一段段可横向推进的旅程。每段路程可以有前景地貌、中景征兆、远景目标和神志污染层，玩家在抵达节点之前就能感到资源压力和叙事风险。

### 6.1 适合做成横向路程段的地点组合

| 路程段 | 适合串联的地点 | 舞台功能 | 主要压力 |
|---|---|---|---|
| 人间出境段 | 中原驿 -> 故王道 -> 废关 | 从可理解秩序过渡到禁令和荒路，作为教程段。 | 车轴、粮草 |
| 河渡水口段 | 旧渡口 -> 井亭 -> 弱水前湾 | 让“水”从补给资源变成规则异常的障碍。 | 粮草、车轴、神志 |
| 坡岭车损段 | 故王道 -> 断车岭 -> 石鼓坡 | 用道路、坡度、车辙和声响表现车轴压力。 | 车轴、神志 |
| 泽畔异象段 | 雷泽浅畔 -> 羽民泽外 -> 赤水外滩 | 以湿地、芦苇、落羽、赤水建立进入异域的第一层压迫。 | 车轴、粮草、神志 |
| 异族边市段 | 青丘旧路 -> 黑齿市 -> 羽民泽外 | 用交易、误解、跟随和被注视感表现语言系统。 | 粮草、神志 |

### 6.3 事件三段式触发时间点

同一个事件可拆成“抵达前、路途中、抵达后”三个触发点，方便横向舞台在玩家尚未进入地点卡之前就制造预兆。

| 触发时间点 | 作用 | 适合事件 | 表现方式 |
|---|---|---|---|
| 抵达地点前 | 预告风险，让玩家提前决定是否继续、绕行或消耗资源探查。 | 车辙分叉、石鼓报时、羽落无影、裂隙前梦 | 路边征兆、远景剪影、随从提醒、地图标记异常。 |
| 路途中 | 制造即时选择，让资源压力发生在两个地点之间。 | 无渡之渡、随从认路、赤水岸骨、空村取粮 | 横屏暂停推进，弹出短事件；选择后继续前进或改道。 |
| 抵达后 | 结算地点核心事件，决定补给、路线、语言理解或污染结果。 | 黑齿交易、断碑读名、赤水岸骨、裂隙前梦 | 地点面板展开，显示可选行动和后续状态变化。 |

### 6.4 神志低时沿途风景如何变得不可信

低神志状态下，横向旅途舞台不应只变暗或变脏，而应让“沿途信息”本身变得可疑。

可用规则：

- 路标不可信：同一块界碑在横向推进中显示不同方向或不同地名。
- 距离不可信：远景目标看似靠近，实际抵达后仍在远处。
- 人数不可信：前景影子、脚印、倒影数量与队伍人数不一致。
- 时间不可信：白天路段出现夜渡征兆，或刚离开的地点重新出现在前方。
- 文字不可信：巫文、价格、禁令会短暂变成玩家队伍成员的名字。
- 资源提示不可信：低神志时可显示“看似更省”的路线，但后续触发额外代价。

原型注意：

- 不建议第一版大量做真假复杂分支，先用 2-3 个可控的“不可信沿途信息”验证体验。
- 低神志污染应影响横向舞台、路线提示和事件文本三处，而不是只换一段描述。
- 污染效果必须仍让玩家能继续决策，不能把可玩信息全部抹掉。

## 7. Web Demo 第一张地图推荐路线顺序

第一版 Web Demo 不建议一次开放全部 20 个地点。建议先用“主路线 + 少量支线”的结构，把横向旅途舞台、三资源压力、事件选择和神志文本都跑通。

### 7.1 推荐主路线

| 阶段 | 推荐地点顺序 | 叙事/玩法作用 | 横向舞台节奏 |
|---|---|---|---|
| 起点 | 中原驿 -> 故王道 -> 废关 | 建立出发、移动消耗、禁行感。 | 画面稳定，远景清楚，文字可信。 |
| 低压段 | 旧渡口 -> 荒禾村 -> 断车岭 | 教玩家粮、车轴和小风险事件。 | 风景开始空旷，前景出现破车、空屋、无主渡具。 |
| 转折段 | 石鼓坡 -> 雷泽浅畔 -> 黑齿市 | 第一次把神志、异族语言和交易误读推到台前。 | 声音、路标和人群开始不稳定，但仍可判断。 |
| 高压段 | 羽民泽外 -> 赤水外滩 -> 巫咸断碑 -> 第二废关 | 地点间变成连续污染，路线、文字和同伴判断都不可靠。 | 前景遮挡增加，远景目标反复出现，地点边界变模糊。 |
| 终点 | 赤水夜渡 -> 九州裂隙 | 第一张地图的收束，导向归返、迷失或进入下一图。 | 横向推进减慢，空间压缩，低神志出现伪路线。 |

### 7.2 可选支线路线

| 支线 | 接入点 | 回流点 | 用途 |
|---|---|---|---|
| 井亭支线 | 旧渡口 | 雷泽浅畔 | 让玩家用粮或神志换取补给/水源判断。 |
| 青丘旧路支线 | 石鼓坡 | 黑齿市或羽民泽外 | 提供较低车轴消耗，但提高语言误读和神志风险。 |
| 弱水前湾支线 | 赤水外滩 | 赤水夜渡 | 作为高风险绕路，适合测试“看似近路”的代价。 |
| 无声林支线 | 羽民泽外 | 第二废关 | 强化低神志沿途不可信，适合作为失败前预警段。 |
| 白骨草甸支线 | 第二废关 | 九州裂隙 | 终局前的资源补偿/污染加深选择。 |

### 7.3 第一版建议锁定的 10 个可玩节点

若 C 线程需要最小可玩版本，建议优先实现：

```text
中原驿 -> 故王道 -> 废关 -> 旧渡口 -> 荒禾村 -> 石鼓坡 -> 雷泽浅畔 -> 黑齿市 -> 赤水外滩 -> 九州裂隙
```

其中“羽民泽外、巫咸断碑、赤水夜渡”可作为第二轮加入的强气质节点；“第二废关、无声林、弱水前湾、白骨草甸”可作为神志污染或失败分支加入。

## 8. 需要用户决策的问题

1. 第一张测试地图是否确认停在“九州裂隙/赤水夜渡”这类边界点，而不进入昆仑、归墟、幽都？
2. 地点命名是否可以采用“古籍名词 + 原创地理后缀”的方式，例如“赤水外滩”“雷泽浅畔”“青丘旧路”？
3. 第一版原型是否允许低神志文本出现“伪提示”，让玩家有机会被错误选项诱导？
4. 异族语言第一版是偏“关键词逐渐解锁”，还是偏“交易和手势判断”，二者若只能先做一个，建议先做关键词解锁。
5. 第一张地图的终局是否需要至少包含一个“可归返但带回污染记录”的非死亡结局方向？
