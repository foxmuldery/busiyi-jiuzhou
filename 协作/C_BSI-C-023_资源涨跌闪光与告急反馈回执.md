# C_BSI-C-023 资源涨跌闪光与告急反馈回执

- 子线程：C / Web Demo 技术原型
- 时间：2026-06-16
- 范围：继续优化状态提醒，让资源变化更像游戏即时反馈，而不是纯文字通知。

## 已实现

- 资源变化时，资源卡会短暂闪光：
  - 正向变化：绿色轻闪。
  - 负向变化：红色轻闪。
- 保留上一轮资源飘字，并与闪光同步出现，例如 `+粮8`、`粮-2`。
- 资源首次跌过警戒线时，加入“车轴告急 / 粮草告急 / 神志告急”舞台浮层队列。
- 更新前端资源版本号，避免浏览器缓存旧样式。

## 验收记录

- 语法检查：`app.js`、`data.js` 通过。
- 差异检查：`index.html`、`app.js`、`styles.css` 通过。
- 浏览器实测：
  - 重开新局后点击“取粮”，粮草 `90 -> 98`，出现 `+粮8`、绿色闪光、“补给完成”浮层，并自动消失。
  - 点击“立刻上路”，粮草 `98 -> 96`，出现 `粮-2`、红色闪光、“立刻上路”结果浮层，并自动消失。
  - 控制台无 error/warn。
