## 1. 主线程快速判断

这份新设定不应被当成普通素材卡处理。它提供了《不思异：九州》此前缺失的四个关键主干：

1. 具体年代与政治局势：永昌年间、烈帝与韩凌君权斗、风宪司成立。
2. 具体主角与搭档：风宪郎陆离、鲛人遗民宛渠。
3. 具体章节路线：沙墟、苍梧、万丘海、玄素山 / 赤霄谷、翠幽檀、洛尘古国。
4. 具体结局逻辑：如实记录民间疾苦、只写盛世奇谈、销毁记录远走。

它和当前 Web Demo 的核心机制高度兼容，尤其是：

- “采风使”可以明确升级为“风宪郎陆离”。
- 当前“颂 / 讽 / 哀 / 谶”可以承接新设定里的“盛世奇谈 / 民间疾苦 / 妖书罪名 / 销毁记录”。
- 当前“王朝案牍”后果可以具体化为烈帝与韩凌君权斗中的政治证据。
- 当前“神志低时文本不可靠”可以接住隐藏剧情“毗骞王 / 盘古轮回 / 九州裂隙”的超自然层。

主线程建议：以新设定作为中长期世界观主干，但不要推翻当前 P0 试玩。第一步只做“设定合并与章节转译”，第二步再由 A/C/D/F 分别评估机制、代码、视觉、音频改造。

## 2. 需要保留的新设定核心

B 线程合并时，以下内容优先保留，不要轻易删改。

### 2.1 主线政治结构

- 风宪司表面用于证明国家安宁、歌功颂德。
- 烈帝暗中借风宪郎陆离记录民间疾苦，反制韩凌君。
- 韩凌君以“妖言惑众”“编造妖书”为潜在罪名威胁陆离。
- 玩家写下的内容不是中性的，它会决定《九州志》成为罪证、盛世奇谈，还是被销毁的禁书。

这能直接增强当前“采风后果”的意义。

### 2.2 主角关系

- 陆离：低调、正直、肺疾、书院记事出身，被烈帝选为风宪郎。
- 宛渠：鲛人遗民，武艺与铁器知识强，潜伏报仇，目标指向韩凌君。
- 二人关系：陆离偏记录与求证，宛渠偏伤痕与复仇；两人能形成选择分歧。

建议 B 线程把陆离 / 宛渠写成“玩家视角内的双声部”，而不是两个普通 NPC。

### 2.3 六章远行路线

新设定章节路线建议作为 P1 / P2 内容主轴：

| 章节 | 地区 | 功能 |
|---|---|---|
| 第一章 | 沙墟 | 教学、采风、民间疾苦、风宪司身份压力 |
| 第二章 | 苍梧 | 边地压迫、韩党地方触手、宛渠身世暗示 |
| 第三章 | 万丘海 | 宛渠身世爆发、珠税与献童、海祭 |
| 第四章 | 玄素山 / 赤霄谷 | 北境空虚、兵备腐坏、冰火对照 |
| 第五章 | 翠幽檀 | 情绪低谷、伐木强征、羁绊深化 |
| 第六章 | 洛尘古国 | 终局真相、古国遗迹、命运 / 记录 / 正史问题 |

### 2.4 三结局结构

新设定的三结局非常适合转成游戏长期目标：

| 结局 | 玩家记录倾向 | 结果 |
|---|---|---|
| A 好结局 | 记录大量民间疾苦和韩党罪证 | 烈帝罢黜韩凌君，宛渠复仇，陆离隐退，《九州志》民间流传 |
| B 坏结局 | 只记录风土民俗和盛世奇谈 | 烈帝灭口陆离，宛渠独自复仇，《九州志》散佚 |
| C 隐藏结局 | 销毁所有记录，不再回京 | 陆离与宛渠隐入荒野，韩党继续把持朝政，民间只余传言 |

这比当前 P0 的“抵达裂隙 / 旅途失败”更完整。建议保留当前 P0 结局作为试玩终点或超自然支线，不直接等同最终正式版结局。

## 3. 与当前 Web Demo 的合并原则

### 3.1 不推翻的内容

当前已经成立的体验不要推翻：

- 三资源仍用 `车轴 / 粮草 / 神志`。
- 底层仍是节点路线，不改成线性视觉小说。
- 九州图 + 横向旅途舞台 + 地点弹窗的结构保留。
- 采风册、地点志、诗引、补给、路遇、低神志文本保留。
- 当前 P0 可试玩路线继续保留为机制测试版本。

### 3.2 需要改名或重新解释的内容

| 当前设定 | 新设定合并建议 |
|---|---|
| 王朝采风使 | 风宪司风宪郎陆离 |
| 抽象王朝 | 永昌朝 / 烈帝朝廷 / 韩党把持的汉廷，需决定是否完全架空 |
| 王朝案牍 | 《九州总志》条目 / 风宪司奏档 / 韩党罪证 |
| 颂 / 讽 / 哀 / 谶 | 保留为写法 UI，但底层记录“粉饰 / 罪证 / 悲悯 / 异兆”倾向 |
| 九州裂隙 | 暂作为隐藏超自然层、洛尘古国或毗骞支线的预告 |
| 黑齿市、青丘等现有地点 | 可暂列为旧 P0 测试地名，后续决定是否归入支线或替换为新六章地理 |

### 3.3 暂不新增资源

新设定带来肺疾、盘缠、罪证、皇权信任、宛渠复仇等变量，但 P0 阶段不建议立刻新增第四资源。

建议处理：

- 陆离肺疾：先作为事件代价或地点状态，不做常驻资源。
- 盘缠百钱：并入粮草 / 补给压力。
- 罪证：先作为采风条目标签，不进入 HUD。
- 宛渠复仇：先作为剧情分支和结局条件。
- 皇帝信任 / 韩党警觉：P1 再评估是否做隐藏变量。

## 4. 关键冲突与需要主创确认的问题

B 线程必须先识别这些冲突，不要直接吞并设定。

### 4.1 年代问题

原文写“公元前156年，汉宣帝染重疾仙逝”，但真实历史中汉宣帝时代与公元前156年不匹配。

需要 B 线程给出两个方案：

1. 完全架空汉朝：保留汉味、永昌、烈帝、韩凌君，但不对真实历史负责。
2. 修正历史锚点：改成年号 / 帝号 / 日期，使其更接近真实历史。

主线程初步倾向：走“架空汉制”，避免被历史准确性牵制。可以写成“旧汉之后，永昌朝”，但不要在对外文案里强调真实公元纪年。

### 4.2 文化锚点风险

原文多个地区标注了现实文化锚点，例如阿拉伯、因纽特、缅老、琉球海女等。

处理原则：

- 这些只能作为内部参考，不要在游戏文本中直说。
- 不要把现实民族特征做成猎奇标签。
- 优先转译为地貌、物产、器物、禁忌、生产方式和被权力压迫的结构。

### 4.3 调性边界

新设定中有强烈苦难、献童、剥削、食人/生啖、少女鳞片等内容。它能提升黑暗志怪力度，但需要控制表达。

B 线程需要区分：

- 可直接进入 P0 文本的内容。
- 只能进入成人向正式版的内容。
- 只能作为暗示或案牍背景的内容。
- 不建议进入游戏的内容。

### 4.4 超自然层级

新设定同时包含政治现实压迫和宇宙轮回超自然。两者都好，但不能互相稀释。

主线程建议：

- 前中期以“风宪司政治采风 + 地方志怪疾苦”为主。
- 低神志、隐藏剧情、洛尘古国、九州裂隙再逐渐打开“世界本身不可靠”的超自然层。
- 毗骞王和盘古轮回不应过早出现。
