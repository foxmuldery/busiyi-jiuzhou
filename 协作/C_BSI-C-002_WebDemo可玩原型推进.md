## 2. 当前实现内容

- 横向山海旅途舞台：假动态长图，不是横版动作游戏。
- 三资源：车轴、粮草、神志。
- 路线选择：路线卡显示起点、终点、风险、资源消耗。
- 事件文本：同一事件按高/中/低神志切换文本。
- 事件选项：选择会改变资源、flag、语言理解度或结局状态。
- 条件路线：赤水到九州裂隙路线需要 flag 或低神志才显明。
- 结果日志：记录路线、资源变化、事件结果。
- 本地存档：localStorage 保存 currentRun、metaProgress、settings。
- 降级策略：动态可关闭，支持 prefers-reduced-motion；素材失败时保留 CSS 兜底。
- 内置校验：浏览器控制台可运行 `window.BSI_PROTOTYPE.validate()`。

## 4. 3 分钟核心循环测试

1. 浏览器打开 `index.html`。
2. 确认顶部显示车轴、粮草、神志。
3. 点击路线“循旧王道西行”。
4. 观察车队推进、背景轻微滚动、资源扣减。
5. 抵达“故王道”后点击任一事件选项。
6. 连续推进到“赤水外滩”，确认“夜近裂隙”初始为尚未显明。
7. 点击“沿骨行路”，确认裂隙路线解锁。
8. 关闭“启用轻量动态”，确认仍可继续选择路线和事件。
