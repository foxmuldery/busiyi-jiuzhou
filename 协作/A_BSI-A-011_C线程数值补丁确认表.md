## 3. `red_bones` 调整最终口径

首版试玩建议采用：

| 项 | 最终口径 |
|---|---|
| 原选项 | `凝视赤水辨路`，神志 -75 |
| 首版处理 | 改为神志 -28，不隐藏 |
| 选项提示 | `禁忌试演 / 神志 -28，触发崩线边缘，解锁骨路` |
| 触发条件 | 保留 `sanityMax: 70` |
| 结果 | 保留 `flag: "followed_bones"` |
| 额外保护 | 若神志 <= 28，点击后进入神志危机，不直接硬失败 |

理由：主线程已给默认建议“首版降为 -28”。这样玩家仍能看到赤水禁忌的强烈代价，但不会被 -75 直接打穿 5 分钟试玩。
