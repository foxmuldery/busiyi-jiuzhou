# 主协调回执：BSI-UX-079 试玩包对外介绍与音频重做清单

日期：2026-06-24

## 1. 本轮目标

用户要求补一段简单介绍，让收到试玩包的人不用理解项目背景、线程制度或内部术语，也能知道《不思异：九州》是什么、试玩时该关注什么。

同时，P0 音频听感验收仍是待人工项。此前工作台可以逐条标记“保留 / 待改 / 弃用”，但没有把待改和弃用项单独整理成下一轮音频任务。

## 2. 已完成改动

### 试玩包对外介绍

已在 `试玩入口.html` 的“复制完整试玩包”内容开头加入简单介绍：

```text
这是一款以《山海经》和古典志怪为灵感的横屏旅途生存文字游戏。你带着车队穿过神秘九州，在每个地点处理怪异遭遇、补给资源、选择下一段路，并尽量保住车轴、粮草和神志。
```

并补充试玩方式：

```text
这次试玩不用研究规则，也不用追求最优解。请像第一次打开手机游戏一样试玩 5 分钟：如果你自然知道下一步点哪里、能感到资源压力、并且愿意再走一站，这版就接近成立；如果哪里像网页、哪里卡住、哪里声音打扰，请直接写出来。
```

### 音频重做清单

已在 `P0音频听感验收工作台.html` 新增：

- `待改/弃用重做清单`
- `复制重做清单`
- `getRevisionItems`
- `buildRevisionPlan`
- `copyRevisionPlan`

当某条音频被标为“待改”或“弃用”时，工作台会自动汇总：

- 音频 ID
- 音频名称
- 当前结论
- 声音层
- 当前状态
- 当前用途
- 循环/短音
- 当前音量
- 听感备注
- 下一轮建议

该清单可直接交给 F 线程或用于下一轮音乐 / 音效生成。

## 4. 边界

- 简单介绍只用于试玩者理解，不作为宣传文案定稿。
- 音频重做清单只服务听感迭代，不代表授权完成。
- `review-pending` 和 `demo-temporary` 仍不能标为正式可发布素材。
