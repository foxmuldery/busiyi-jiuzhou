# C_BSI-C-016 左下角展开按钮修复回执

## 问题

用户反馈：左下角“展开”点击后没有反应。

## 判断

按钮绑定存在，问题主要在小横屏 CSS：左下角“当前遭遇”卡片展开后仍被限制在原小格子中，视觉上几乎没有变化。

## 已修复

1. `setTextPanelExpanded` 改为互斥展开：打开地点志时收起当前遭遇，打开当前遭遇时收起地点志。
2. 小横屏下：
   - 地点志展开后跨到下方区域。
   - 当前遭遇展开后跨两列两行，显示更大的文本/插图区域。
   - 展开按钮提高层级，减少被覆盖或误触风险。

## 验证

- `app.js` 语法检查通过。
- `data.js` 语法检查通过。
- `git diff --check` 通过。
