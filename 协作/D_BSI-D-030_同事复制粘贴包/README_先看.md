# D_BSI-D-030 同事复制粘贴包说明

用途：给同事在 ChatGPT 网页版或其他生图工具里逐条生成《不思异：九州》下一轮 24 张图。

## 使用顺序

1. 打开 `参考图` 文件夹。
2. 每条 Prompt 会写明要上传哪些参考图，例如 `REF-01`、`REF-02`。
3. 先上传对应参考图，再复制 `D_BSI-D-030_24张Prompt_同事复制版.md` 里的整段 Prompt。
4. 生成文件按 Prompt 顶部的文件名保存。

## 参考图标记

| 编号 | 文件 | 用途 |
|---|---|---|
| `REF-01` | `参考图/REF-01_共同风格_横向山海旅途舞台.png` | 所有图默认共同风格参考 |
| `REF-02` | `参考图/REF-02_废关古道_残关路面.png` | 古道、废关、荒原、关门相关 |
| `REF-03` | `参考图/REF-03_雷泽赤水_水岸湿地.png` | 雷泽、赤水、水岸、泥泡相关 |
| `REF-04` | `参考图/REF-04_归墟裂隙_暗水终点.png` | 归墟、裂隙、梦境、神志危机相关 |

## 三条硬规则

- 背景图不要车队、不要 UI、不要文字。
- 事件弹窗优先 `4:3`，物件路遇可用 `1:1`。
- 需要抠图的 UI 小图标用纯绿色背景，不要透明棋盘格。
