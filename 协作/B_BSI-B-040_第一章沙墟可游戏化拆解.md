# BSI-B-040 第一章沙墟可游戏化拆解

## 结论

第一章应优先拆《吾身化种》，作为下一轮 Web Demo 最适合承载的新主线纵切片。它能同时展示采风身份、韩党粉饰、民间疾苦、陆离/宛渠关系、三资源压力和“颂 / 讽 / 哀 / 谶”的后果，不需要新增数值系统。建议做成 6 个地点节点、5 个地点事件、5 个半途路遇，以“果枣盛世”逐步反转为“人身化种”的惊悚现实。
