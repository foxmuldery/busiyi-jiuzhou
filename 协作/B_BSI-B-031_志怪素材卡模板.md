# BSI-B-031 志怪素材卡模板

用途：给志怪内容协作者、编剧、资料整理和游戏转译人员使用。  
原则：只写文学素材和故事潜力，不要求填写游戏数值。

复制下面模板，一张卡只写一个素材。

```markdown
# 志怪素材卡：素材短名

## 1. 基本信息

| 字段 | 内容 |
| --- | --- |
| 素材编号 | SYYS-000 |
| 素材短名 | 例如：食梦井 |
| 原项目出处 | 《山野逸事》第三部，第几集/第几场/哪份设定 |
| 古典来源 | 例如：《搜神记》《太平广记》《山海经》或民间传说；不确定就写“待查” |
| 来源可信度 | 确认 / 待查 / 原创改写 / 多来源混合 |
| 当前状态 | draft / ready-for-review / needs-source-check |

## 2. 一句话故事核

用一句话说明这个素材为什么有意思。

> 例：一口井会吞掉人的梦，旅人喝水后忘记自己为何西行。

## 3. 志怪意象

- 怪物 / 人物：

- 仪式：

## 4. 情绪与气质

- 诡谲
- 神圣

- 民间
- 梦魇
- 诱惑
- 失路
- 其他：

写玩家抵达或路遇时第一眼看到的东西。

> 例：井沿没有水痕，只有许多低头睡着的人影，影子的嘴里还在说梦话。

- 行为 1：

从故事角度写，不填具体数字。

写 1 到 3 句，像玩家在图鉴或地点志里看到的记录。

> 

### 事件正文口吻

写 1 到 3 句，像弹窗里看到的遭遇描述。

> 

### 低神志变体

神志低时，这个素材会变得哪里不可信、更恐怖或更疯狂？

可多选：

- 地点志

```

## 填写提醒

- 不要写长篇故事梗概。
- 不要在一张卡里塞多个典故。
- 不要直接复制古籍大段原文或现代译文。
- 优先写“玩家能遇见什么、能做什么、会失去什么”。
- 如果素材很美但不能选择，先标为地点志或图鉴，不急着做事件。
