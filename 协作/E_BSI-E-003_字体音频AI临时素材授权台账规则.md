# BSI-E-003 阻断级素材风险检查

> 子线程：E 商业合规与竞品研究顾问  
> 任务：只检查会不会明显阻断 Web Demo 跑起来  
> 日期：2026-06-14  
> 结论性质：产品侧风险判断，不构成法律意见  
> 当前原则：先让 Web Demo 跑起来，细节合规后置；不讨论宣传，不 git commit / push

## 0. 当前执行原则

本阶段只做阻断级判断：

```text
明显不能进 Web Demo -> 标红阻断
可以临时跑起来 -> 标为 Demo 临时
正式发布、商用、平台、宣传细节 -> 后置复核
```

不要因为字体、音乐、图片、视频、AI 临时素材的细节问题阻塞 C/D/F 做可玩原型。只有下面红线命中时，才要求替换或禁用。

## 1. 红线总表

| 类型 | 阻断级红线 | 处理 |
|---|---|---|
| 字体 | 来源不明字体文件、盗版字体包、明确非商用/禁止网页嵌入字体 | 不进 Demo，改用系统字体或 OFL 字体 |
| 音乐/音效 | 竞品、影视、游戏、音乐平台、短视频平台直接下载/截取音频 | 不进 Demo，静音或用自制/AI 临时音频 |
| 图片/视频 | 竞品截图、影视截图、游戏截图、现代插画、来源不明网图 | 不进 Demo，改用自制占位/AI 临时素材 |
| AI 素材 | 提示语或结果明显指向具体竞品、画师、影视、音乐人、品牌、角色 | 不进 Demo，重生成更抽象版本 |
| 代码/库 | 许可证明确禁止商业或再分发，或要求开源整站但项目不准备遵守 | 不引入，换 MIT/BSD/Apache 等低风险库 |
| 《山海经》资料 | 现代译文、百科段落、现代插画直接进入游戏正文或画面 | 不进 Demo，改为自写文本/自制占位 |

## 2. 可以先临时使用

以下不阻断 Web Demo，但必须标为 `Demo 临时`，后续统一复核：

- 系统字体栈。
- OFL 字体，例如思源宋体、Noto Serif CJK、霞鹜文楷等。
- 自制几何占位图、线稿、剪影、纯色块。
- AI 生成图像/视频/音频，只要没有明显模仿具体作品或人物。
- 用户上传的音乐候选，只能作为内部 Demo 临时候选，不能标为正式可用。
- 自研代码和 MIT/BSD/Apache/MIT-like 前端库。

## 3. 最小台账字段

为了不拖慢原型，只记录 6 个字段：

| 字段 | 示例 |
|---|---|
| 资产 ID | `FONT-001`、`MUS-TMP-001`、`AI-BG-001` |
| 文件路径 | 素材实际路径 |

| 来源一句话 | 系统字体 / OFL / AI 生成 / 用户上传 / 自制 / 第三方库 |
| 当前状态 | Demo 临时 / 阻断禁用 / 后置复核 |
| 阻断原因 | 只有标红时填写 |

## 4. 给 C/D/F 的快速判断

### 字体

- 默认用系统字体栈，不阻断。
- 想要古籍感，先用 OFL 字体，不阻断。
- 来路不明的古风字体、书法字体、字体包，阻断。

### 音频

- 没有音频也能跑，先静音不阻断。
- AI 临时音频可进内部 Demo，但标 `Demo 临时`。
- 任何从竞品、影视、音乐平台、短视频平台拿来的音频，阻断。

### 图像 / 视频

- 自制占位、AI 临时素材、几何图形、剪影，不阻断。
- 竞品截图、影视截图、现代插画、百科图片、来源不明网图，阻断。
- 视频太大、遮挡 UI、影响点击，属于体验/技术阻断，交给 C/D 处理；E 只管来源红线。

### AI 素材

- 提示语不写具体作品、画师、音乐人、品牌、角色名，就先允许 Demo 临时使用。
- 生成结果看起来像具体竞品、影视、画师或现代图鉴，阻断并重做。
- 正式可用状态以后再判断，现在不展开。

### 代码 / 开源库

- React、TypeScript、Vite、MIT/BSD/Apache 类库，不阻断。
- ChoiceScript 这类明确非商业限制的库，阻断。
- 不明许可证、无 LICENSE、直接 fork 完整游戏换皮，阻断。

## 5. 当前红线提醒

- 不要为了美术效果引入来源不明字体。
- 不要为了氛围引入网上下载的音乐或音效。
- 不要把现代《山海经》译文、百科、插画直接放进游戏。
- 不要复制 Out There、FAR、DREDGE、Sunless Sea、Roadwarden 等竞品的 UI、文案、音乐、镜头或图像。
- 不要让“授权未确认”阻挡原型，先标 `Demo 临时`；只有“明显来源不行”才阻断。
