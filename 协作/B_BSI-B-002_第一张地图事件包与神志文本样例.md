# BSI-B-002 第一张地图事件包与神志文本样例

> 子线程 B：山海经世界观与叙事主创  
> 任务目标：为 Web Demo 第一张地图提供可进入原型的数据化事件草案、神志文本样例和跨线程接口。  
> 边界：本文件不是最终商业文案，不决定数值公式，不修改 A/C/D/E/F 交付文件。

## 1. 本轮产物定位

本文件服务第一版 Web Demo，重点不是写完正式剧情，而是让 C 线程可以把事件放进原型，让 A 线程可以给事件套权重和死局防护。

内容原则：

- 每个事件都能落到地点或路程段。
- 每个事件都有 2-3 个选择，选择会改变车轴、粮、神志或线索状态。
- 低神志文本可以异常、疯狂，但必须保留至少一个可玩信息，例如主要风险、方向、代价或可选行动。
- “粮/粮草”在叙事中可写粮草，原型字段建议统一为 `grain`。
- 事件代价先用小/中/大标注，具体数值交给 A 线程。

## 2. Web Demo 第一张地图推荐路线顺序

### 2.1 主路线

| 阶段 | 地点顺序 | 核心体验 | 建议事件强度 |
|---|---|---|---|
| 起点 | 中原驿 | 让玩家理解出发、补给、修车和记录。 | 低 |
| 低压段 | 故王道 -> 废关 -> 旧渡口 -> 荒禾村 | 用清楚风险教会三资源含义。 | 低到中 |
| 转折段 | 断车岭 -> 石鼓坡 -> 雷泽浅畔 -> 黑齿市 | 第一次出现巫文、异族语言和神志文本差异。 | 中 |
| 高压段 | 羽民泽外 -> 赤水外滩 -> 巫咸断碑 -> 第二废关 | 让地点之间变成连续污染，路线提示开始不完全可信。 | 中到高 |
| 终点 | 赤水夜渡 -> 九州裂隙 | 收束为归返、迷失、进入下一图三类方向。 | 高 |

### 2.2 第一版最小可玩路线

若第一版只做 8-10 个节点，建议用：

```text
中原驿 -> 故王道 -> 废关 -> 旧渡口 -> 荒禾村 -> 石鼓坡 -> 雷泽浅畔 -> 黑齿市 -> 赤水外滩 -> 九州裂隙
```

这个顺序能覆盖：

- 起点补给和出发。
- 车轴压力。
- 粮草补给诱惑。
- 神志事件。
- 异族语言事件。
- 禁地终点。

## 3. 给 A 线程可用的事件风险标签

以下标签只表达事件性质，不决定具体权重。

| 标签 | 含义 | A 线程可用方式 |
|---|---|---|
| `pool_risk` | 风险事件，主要制造损耗和选择压力。 | 进入风险事件池。 |
| `pool_supply` | 补给事件，能恢复或交换资源。 | 进入补给事件池，受坏运气保护影响。 |
| `pool_clue` | 线索事件，给路线、语言或隐藏信息。 | 可作为关键路线兜底，不应完全随机缺失。 |
| `pool_item` | 器物事件，影响车轴、交易或后续选项。 | 后续若有器物系统可接入。 |
| `pool_sanity` | 神志事件，读碑、梦、幻路、异常文本。 | 需限制连续触发，避免早期崩解。 |
| `pool_rest` | 喘息事件，低压复盘、休整或小恢复。 | 三资源低时可提高权重。 |
| `risk_low` | 损耗低，信息可靠。 | 起点和低压段主用。 |
| `risk_mid` | 损耗中等，有交换和误读。 | 转折段主用。 |
| `risk_high` | 损耗高，可能改变路线或结局。 | 高压段和终点限制使用。 |
| `bias_axle` | 主要压车轴。 | 用于检查车轴失败占比。 |
| `bias_grain` | 主要压粮。 | 用于检查粮失败占比。 |
| `bias_sanity` | 主要压神志。 | 用于检查神志失败占比。 |
| `bad_luck_guard` | 资源低时应降权或替换。 | 接入坏运气保护。 |
| `rescue_candidate` | 可作为低资源补救事件。 | 危急线以下可提高权重。 |

## 4. 14 个可进原型的事件草案

资源代价/收益采用相对标注：

- 小：轻微影响，适合教程和低风险段。
- 中：明显影响，需要玩家权衡。
- 大：高风险，不应连续强制触发。

### E01 出驿问路

- 地点：中原驿
- 触发条件：首局开始或玩家首次出发前。
- 事件描述：驿中尚有人间秩序，玩家可以在修车、购粮、问路之间做出第一组资源选择。
- 选择：
  - 买足干粮：粮 +中，车轴不变，神志不变。
  - 检查车轴：车轴 +小，粮 -小，神志不变。
  - 只问西路：获得 `clue_old_road_warning`，神志 -小。
- 资源代价/收益：补给或车轴小修；问路换线索。
- 神志影响：低，主要用于教学。
- 风险标签：`pool_supply`, `pool_clue`, `risk_low`, `rescue_candidate`

### E02 禁令残碑

- 地点：故王道 / 废关前路
- 触发条件：第一次进入废关前，或神志低于中线时在路途中再次触发。
- 事件描述：路边旧碑刻着禁止西行的残令，残字与车辙方向不一致。
- 选择：
  - 按车辙继续：车轴 -小，粮 -小，获得正常前进。
  - 停车辨碑：神志 -小，获得 `clue_pass_reverse_mark`。
  - 绕开碑路：车轴 -中，粮 -小，降低本次神志风险。
- 资源代价/收益：在车轴消耗和神志线索间交换。
- 神志影响：中；低神志时碑文可能显示伪方向，但仍保留“继续/绕行”信息。
- 风险标签：`pool_clue`, `pool_sanity`, `risk_low`, `bias_sanity`

### E03 废关修辐

- 地点：废关
- 触发条件：抵达废关后，车轴低于安全线或首次通过废关。
- 事件描述：废关下堆着旧车碎件，可拆取修辐，但旧令和断木会拖慢队伍。
- 选择：
  - 拆旧车修辐：车轴 +中，粮 -小。
  - 搜关楼找路：获得 `clue_abandoned_pass_side_path`，神志 -小。
  - 立刻通过：车轴 -小，跳过收益。
- 资源代价/收益：车轴补救点，也可换路线线索。
- 神志影响：低到中。
- 风险标签：`pool_supply`, `pool_item`, `risk_low`, `bias_axle`, `rescue_candidate`

### E04 无渡之渡

- 地点：旧渡口
- 触发条件：从废关进入旧渡口，或选择水路支线。
- 事件描述：渡船在岸，船绳未断，但没有渡子；水面看似平静，渡具像刚被人放下。
- 选择：
  - 修船强渡：车轴 -中，粮 -小，推进到对岸。
  - 沿岸找浅处：粮 -中，神志 -小，可能获得低风险回路。
  - 等到夜潮：粮 -小，神志 -中，可能解锁 `route_night_shoal`。
- 资源代价/收益：车轴、粮、神志三路交换。
- 神志影响：中；夜潮选项提高后续污染事件权重。
- 风险标签：`pool_risk`, `pool_clue`, `risk_mid`, `bias_axle`, `bias_sanity`

### E05 空村晾粮

- 地点：荒禾村
- 触发条件：粮低于预警线时高权重；正常抵达时中权重。
- 事件描述：村中无人，院中晾着谷物，灶灰仍温；粮食可取，但村子像在等人回来。
- 选择：
  - 取走谷物：粮 +中，神志 -小，添加 `flag_took_empty_village_grain`。
  - 留下一半并祭告：粮 +小，神志 +小或不变。
  - 不入村：无收益，粮 -小，降低后续污染风险。
- 资源代价/收益：早期主要补粮事件。
- 神志影响：低到中；低神志时粮袋可能被描述成“会呼吸”，但仍明确可取粮。
- 风险标签：`pool_supply`, `pool_sanity`, `risk_low`, `bias_grain`, `rescue_candidate`

### E06 断辐坡

- 地点：断车岭
- 触发条件：进入坡岭路程段；车轴低时高权重。
- 事件描述：坡上旧车辐密布，行路越慢越稳，越快越省粮。
- 选择：
  - 慢行护车：粮 -中，车轴 -小。
  - 催车上坡：粮 -小，车轴 -中。
  - 拾旧辐加固：粮 -小，神志 -小，车轴 +小。
- 资源代价/收益：明确教玩家车轴和粮的互换。
- 神志影响：低。
- 风险标签：`pool_risk`, `pool_item`, `risk_low`, `bias_axle`

### E07 石鼓三响

- 地点：石鼓坡
- 触发条件：首次抵达石鼓坡；或玩家已获得任一巫文线索。
- 事件描述：坡上石鼓无手自响，三声之后，队伍对时间和距离的判断开始分叉。
- 选择：
  - 记录鼓声：神志 -小，获得 `clue_stone_drum_rhythm`。
  - 敲鼓回应：神志 -中，可能解锁巫文相关选项。
  - 掩耳离开：粮 -小，避免本次神志损耗。
- 资源代价/收益：巫文/神谕入口。
- 神志影响：中；低神志时鼓声会提供不完全可靠的路线提示。
- 风险标签：`pool_sanity`, `pool_clue`, `risk_mid`, `bias_sanity`, `bad_luck_guard`

### E08 雷泽陷车

- 地点：雷泽浅畔
- 触发条件：进入泽畔路程段；雨雾/低神志状态下权重提高。
- 事件描述：泽水不深，但黑泥吞轮，远雷贴地滚动，车队必须选择脱困方式。
- 选择：
  - 众人推车：车轴 -小，粮 -中。
  - 拆载减重：车轴 +小或避免车轴损耗，粮 -小，失去少量补给标记。
  - 顺雷声绕行：车轴 -中，神志 -小，可能避开下一次风险事件。
- 资源代价/收益：中段车轴压力。
- 神志影响：低到中。
- 风险标签：`pool_risk`, `risk_mid`, `bias_axle`, `bias_grain`

### E09 黑齿换物

- 地点：黑齿市
- 触发条件：首次进入异族边市；若已有语言线索则显示额外提示。
- 事件描述：黑齿市午时忽现，交易者以手势指向粮袋、车轴和来路，价格不完全能懂。
- 选择：
  - 以粮换车具：粮 -中，车轴 +中。
  - 以旧器换路标：失去器物或线索标记，获得 `clue_black_teeth_route_sign`。
  - 拒绝交易离开：无收益，神志 +小或不变，可能错过支线。
- 资源代价/收益：交易与语言理解事件。
- 神志影响：中；低神志时价格符号会伪装成地名，但“换车具/换路标/离开”仍清楚。
- 风险标签：`pool_supply`, `pool_clue`, `risk_mid`, `bias_grain`, `bias_sanity`

### E10 羽落无影

- 地点：羽民泽外
- 触发条件：从雷泽或黑齿市进入泽外路段；拥有 `clue_black_teeth_route_sign` 时可降低误读。
- 事件描述：芦苇间落下轻羽，地上没有羽影；随从听见上方有语言，但看不见说话者。
- 选择：
  - 低头疾行：车轴 -小，粮 -小，避开交流。
  - 以布接羽：神志 -中，获得 `clue_feather_speech_token`。
  - 呼喊回应：神志 -小，可能触发语言误读或支线。
- 资源代价/收益：语言线索和神志交换。
- 神志影响：中到高。
- 风险标签：`pool_clue`, `pool_sanity`, `risk_mid`, `bias_sanity`, `bad_luck_guard`

### E11 赤水岸骨

- 地点：赤水外滩
- 触发条件：首次抵达赤水外滩；低神志或夜行时权重提高。
- 事件描述：岸边白骨排成近似路标的形状，白骨间的草朝同一方向伏下。
- 选择：
  - 依骨行路：车轴 -小，神志 -中，可能提前接近夜渡。
  - 掩骨祭拜：粮 -小，神志 +小，降低本段高风险。
  - 远离水岸：车轴 -中，粮 -小，避开赤水污染。
- 资源代价/收益：禁地入口的路线选择。
- 神志影响：中到高；低神志时骨会“数人”，但仍保留三种行动方向。
- 风险标签：`pool_sanity`, `pool_risk`, `risk_high`, `bias_sanity`, `bad_luck_guard`

### E12 断碑读名

- 地点：巫咸断碑
- 触发条件：拥有任一巫文/鼓声线索，或从赤水外滩进入断碑路段。
- 事件描述：断碑只剩半截，缺字会随着读者的判断自行补全。
- 选择：
  - 读碑：神志 -中，获得隐藏路线或禁忌提示。
  - 拓碑带走：粮 -小，神志 -小，获得 `item_stele_rubbing`。
  - 毁去碑面：车轴 -小，神志 +小，失去后续巫文线索。
- 资源代价/收益：高风险线索事件。
- 神志影响：高；低神志读碑可出现伪提示，但不能隐藏“读/拓/毁”的行动后果类型。
- 风险标签：`pool_sanity`, `pool_clue`, `risk_high`, `bias_sanity`, `bad_luck_guard`

### E13 第二废关

- 地点：第二废关
- 触发条件：已经过废关；进入高压段后触发。
- 事件描述：前方废关与来路废关几乎相同，门额方向倒置，队伍怀疑自己回到了原处。
- 选择：
  - 按旧法通过：车轴 -小，神志 -中，可能直接进入赤水夜渡。
  - 对照旧记录：粮 -小，神志 +小，确认是否为环路。
  - 拆门为材：车轴 +小，神志 -小，添加 `flag_broke_second_pass`。
- 资源代价/收益：高压段的认知校验和车轴补救。
- 神志影响：中到高。
- 风险标签：`pool_sanity`, `pool_item`, `risk_high`, `bias_sanity`, `rescue_candidate`

### E14 裂隙前梦

- 地点：九州裂隙
- 触发条件：抵达终点前夜；或神志低于危急线时提前触发。
- 事件描述：所有人梦见同一张地图，地图上有一条更近的路，也有一条回头路。
- 选择：
  - 按梦中近路走：神志 -大，可能进入隐藏路线或迷失结局。
  - 烧掉梦图：粮 -小，神志 +小，进入稳定终局判断。
  - 对照旧图归返：车轴 -中，粮 -中，获得归返结局方向。
- 资源代价/收益：终点分流事件。
- 神志影响：高；低神志仍必须明确“近路危险、烧图稳定、归返耗资源”。
- 风险标签：`pool_sanity`, `pool_clue`, `risk_high`, `bias_sanity`, `bad_luck_guard`

## 5. 神志高/中/低三层文本样例

以下文本仍是原型样例，不是最终正式文案。低神志版本保留可玩信息。

### 5.1 E07 石鼓三响

高神志：

> 坡上有三面石鼓。风过时，鼓腹自鸣三声。随从说此地宜速过，也可记下鼓声，日后辨路。

中神志：

> 石鼓响了三声，车轮却像多走了一日。你记得来路在东，鼓声却把东敲成了西。

低神志：

> 鼓在数路。一声是走，二声是绕，三声是别回头。你仍可记声、回应，或立刻离开。

### 5.2 E09 黑齿换物

高神志：

> 市人齿黑，言语难辨。他们指向粮袋，又指向车轴，似乎愿以粮换车具，或以旧器换路标。

中神志：

> 他们不说价格，只把黑齿露给你看。你听不懂句子，却听见“粮”“路”“明日”三个词。

低神志：

> 市不是市，是一排会笑的门。左边吃粮吐车具，右边吃旧物吐路标。你也可以不进门。

### 5.3 E11 赤水岸骨

高神志：

> 赤水岸边有白骨排列，似为旧行旅留下的标记。草向下游伏倒，远处似有浅滩。

中神志：

> 白骨不像死物，更像刚被摆好。草尖一齐指向夜色深处，随从不愿再看水面。

低神志：

> 骨在指路，也在点名。沿骨走会近，祭骨会稳，离骨走会远。水等到夜里才开口。

### 5.4 E12 断碑读名

高神志：

> 断碑半埋土中，缺字甚多。若细读，或可辨出前方禁忌；若拓下碑文，或可留待后来参照。

中神志：

> 碑上缺字会自己补齐。你刚看见“赤水不可夜渡”，下一眼又像写着“夜渡可归”。

低神志：

> 碑没有字，字在你眼里补。读会伤神，拓会带走它，毁碑可止住它。它正在写你的名。

### 5.5 E14 裂隙前梦

高神志：

### 6.3 资源变化枚举建议

为避免 B 线程直接定数值，建议先用枚举给 A/C 映射：

| 枚举 | 含义 |
|---|---|
| `gain_small` | 小幅恢复 |
| `gain_medium` | 中幅恢复 |
| `gain_large` | 大幅恢复，需谨慎 |
| `loss_small` | 小幅消耗 |
| `loss_medium` | 中幅消耗 |
| `loss_large` | 大幅消耗，避免连续强制 |
| `none` | 无变化 |
| `uncertain` | 低神志或未识语言时显示为模糊代价 |

### 6.4 事件触发时间枚举建议

| timing | 含义 | 舞台表现 |
|---|---|---|
| `pre_arrival` | 抵达地点前 | 横向舞台出现路边征兆或远景提示。 |
| `mid_route` | 路途中 | 推进暂停，弹出短事件，选择后继续或改道。 |
| `arrival` | 抵达后 | 展开地点事件面板，进行资源和线索结算。 |
| `night` | 夜间/等待后 | 用于赤水夜渡、裂隙前梦等事件。 |
