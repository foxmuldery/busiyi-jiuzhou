# 《不思异：九州》采风使开场动画 旁白单独稿

> 用途：配合 10 张动态漫画开场图使用。  
> 口吻：王朝诏命感 + 惊悚叙事感，不用百科腔，不解释机制名。  
> 建议：每镜头 6-8 秒；每镜头只配一个短句，留出画面呼吸。  

## 1. 10 镜头逐段旁白

| 镜头 | 对应画面 | 旁白 |
| --- | --- | --- |
| 01 | 王朝档案库，官员展开九州舆图 | 九州入图，万风未录。 |
| 02 | 官员授命，采风使接过册牍 | 朝命西行，采风成册。 |
| 03 | 主角离开宫城，整理箱笼与行装 | 一册诗简，载你出关。 |
| 04 | 车队陷入泥路，车轴、粮袋、疲惫同行者 | 车轴欲折，粮影渐薄。 |
| 05 | 官员、百姓、异象三方声音压向主角 | 官声、民声、山川皆有声。 |
| 06 | 官员递贿，饥民求告，主角夹在中间 | 袖中有金，阶前有饥。 |
| 07 | 祭所与粮争，主角持笔介入冲突 | 一笔落下，众命相牵。 |
| 08 | 桌上竹简、笔、镜面与城池倒影 | 颂讽哀谶，皆可治城。 |
| 09 | 诗简回到王朝，城池命运在倒影中变化 | 诗入朝堂，城有生死。 |
| 10 | 主角站在裂开的九州舆图上 | 奉命采风，写下即判。 |

## 2. 完整连读版

```text
九州入图，万风未录。

朝命西行，采风成册。

一册诗简，载你出关。

车轴欲折，粮影渐薄。

官声、民声、山川皆有声。

袖中有金，阶前有饥。

一笔落下，众命相牵。

颂讽哀谶，皆可治城。

诗入朝堂，城有生死。

奉命采风，写下即判。
```

## 4. 配音节奏建议

- 每句建议 2-3 秒念完，剩余时间给画面和音效。
- 01-03 镜头：低声、克制，像旧档开封。
- 04-07 镜头：语速略慢，压出路途损耗和道德选择。
- 08-10 镜头：更冷、更轻，最后一句可留半秒空拍。
