## 6. 给 C 线程的小接入建议

1. 第一轮只接 `VID-OV-CLOUD-001`、`VID-OV-SAND-001`、`VID-OV-CRACK-001` 三个视频叠层。
2. 所有视频必须有 poster 或静态图 fallback，动态开关关闭时不加载视频。
3. 视频层只进舞台/地图区域，不覆盖顶部 HUD、底部事件文本、选择按钮、行囊和日志。
4. 小横屏下优先禁用背景类视频，只保留低透明云雾或直接回退静态叠层。
5. UI 切片先试装 1 个资源条底板、1 个选择按钮底板、1 个事件底板，不要一次全换。
