> 子线程：E，商业合规与竞品研究顾问  
> 日期：2026-06-16  
> 范围：阻断级风险复核；不阻塞内部 Demo 继续测试

## 结论

当前试玩版可以继续内部测试。未发现必须暂停 Web Demo 的阻断级风险。

本轮判断按三类处理：

1. 可以继续内部测试：玩法循环、原创文本、当前可用素材目录、OFL 字体、程序合成音效、Demo 临时音乐与 AI 临时视觉素材。
2. 发布前必须替换或复核：用户上传/AI 音乐候选、车队假透明素材、品牌台标、AI 生成图像/视频/音频、古诗/API 来源、所有 `待复核素材`。
3. 禁止使用/禁止表达：竞品 UI/文案/图像/音效/音乐、现代《山海经》译文/百科/插画、来源不明素材、明确非商业代码库、Out There / Darkest Dungeon / FAR / 西部旅行的具体表达。

核心原则：

```text
先让内部 Demo 跑起来。
Demo 临时不等于正式可用。
只有明显来源不行、明显像竞品、明显不能授权的内容才阻断。
```

## 禁止使用 / 禁止表达

### 禁止使用

- 竞品截图、商店图、logo、UI 图标、角色、宣传图。
- 竞品、影视、游戏、短视频平台、音乐平台截取的音乐或音效。
- 来源不明网图、现代山海经图鉴、现代插画、影视/漫画/游戏截图。
- 现代《山海经》白话译文、百科段落、公众号/资料站段落直接进入游戏正文。
- 直接 fork 完整开源游戏换皮。
- ChoiceScript 等明确禁止商业或再分发条件不适配的库。
- 无 LICENSE、许可证不明、或要求开源整站但项目不准备遵守的代码库。
- AI 提示词点名具体竞品、影视、画师、音乐人、品牌、角色名生成的素材。

### 禁止表达

- 不写“山海经版 Out There / 中国版 Out There / Out There 精神续作”。
- 不写“山海经版 Darkest Dungeon / 中国版 DD”。
- 不写“FAR-like 载具旅途 / 类 FAR 横版车队”。
- 不把游戏描述成西部车队换皮。
- 不用星图、星球节点、发光航线、雷达扫描、科幻 HUD 表达九州图。
- 不用哥特英雄队列、火把压力条、Affliction/Virtue 式精神病症表达神志。
- 不用西部马车、牛仔、枪械、赏金、酒馆木牌、口琴等西部符号。

## 需要 C 线程实现的内容

本轮没有阻断 C 线程继续内部测试的风险项。若 C 线程要做最小落地，可以只注意：

1. 保持音频默认关闭或用户点击后启用；缺音效时静默可玩。
2. 音频状态仍显示“临时音乐/临时音效”，不要在 UI 或调试信息里写成正式素材。
3. 低神志星象/污染层只在污染状态出现，不作为常态路线/节点连接视觉。
4. 若替换车队层，优先使用真透明 PNG；不要把背景自带车队图当舞台背景。
5. 若接入新素材，至少在数据或台账中保留 `Demo 临时 / 待复核 / 禁止使用` 状态。
