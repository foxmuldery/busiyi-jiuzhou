# 《不思异：九州》音乐音效资产区

这个目录用于管理 Web Demo 和后续版本的音乐、音效、环境声、音频提示语、授权记录与待复核素材。

## 目录结构

| 目录 | 用途 |
|---|---|
| `00_音频台账` | 音频资产清单、来源、授权、使用状态 |
| `01_音乐方向` | 背景音乐、区域音乐、动态分层方向 |
| `02_音效方向` | UI、资源、路线、事件、告警音效方向 |
| `03_生成提示语` | AI 音乐/音效生成提示语与参数记录 |
| `04_待复核素材` | 未确认授权、未验收、待筛选音频 |
| `05_可用素材` | 已通过主线程验收的可用音频 |

## 使用原则

1. 不上传未确认授权的音乐或采样。
2. 不模仿具体竞品、影视、游戏的旋律、音色或标志性音效。
3. AI 生成音频必须记录工具、日期、提示语、用途和授权状态。
4. 第一版优先做短循环和交互音效，不追求完整 OST。
5. 音频文件进入 Git 前必须经过主线程确认，避免仓库膨胀。
