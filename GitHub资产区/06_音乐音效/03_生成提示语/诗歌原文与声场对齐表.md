# 诗歌原文与声场对齐表

> 项目：《不思异：九州》Web Demo  
> 日期：2026-06-18  
> 用途：把 `data.js` 的固定诗引与音乐、环境声、事件短音乐、神志污染层对齐。  
> 边界：本文件是声音现场工作表，不导入新音频，不把任何音乐标为正式可用。

## 0. 现场原则

1. 诗歌原文只作为叙事锚点，不做主音乐歌词。
2. 音乐生成提示语仍保持“不要歌词、不要清晰人声、不要可辨认咒语”。
3. 如果要做朗读，建议独立成很短的 `VOICE-VERSE-*` 资产，触发时长 1.5-4 秒，音量低于事件正文。
4. 如果要做神志污染，可以把诗句做成不可辨的远处气息、倒放纹理或断片回声，但不能让玩家听清完整句子。
5. 正式发布前，所有原文节选应按统一底本二次校勘；当前为 Codex 工作版，供音频现场先对方向。

## 4. 生成提示语补充模板

如果某段音乐需要带诗意锚点，不要把原文写成歌词。可使用以下模板：

```text
参考诗意锚点：{诗句原文}。不要出现歌词，不要可辨认朗读，不要清晰人声。只把这句的情绪转成声场：{声场描述}。音乐保持低音量、留白、古典、克制，不能压过文字阅读。
```

如果要做独立短朗读资产，可使用：

```text
录制一条 1.5-4 秒的极轻古辞短句朗读，内容为“{诗句原文}”。语气克制、像行旅日志开头，不戏剧化，不吟唱，不拖腔。尾音自然落下，留 300ms 空气尾。只用于事件说明开头，不进入背景音乐。
```

如果要做低神志纹理，可使用：

```text
用“{诗句原文}”作为不可辨识的低语纹理来源，但最终声音不能让人听清任何完整词句。处理成远处、倒放、断片、低频被水声遮住的质感，音量低，不能压过文字阅读。
```
