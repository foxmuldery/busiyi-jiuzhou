# 第一批音频生成提示语

## 通用禁止项

- 不引用具体作曲家、影视、游戏、竞品或音乐厂牌。
- 不要求“像某某作品”。
- 不使用清晰歌词、人声唱词或可辨认咒语。
- 不使用未授权采样包。
- 不要史诗管弦、仙侠热血、现代电子舞曲、恐怖片跳吓。

## MUS-001 荒外低频底乐

```text
生成一段适合中国古代神秘九州远行游戏的低存在感背景音乐，时长 45 秒，可无缝循环。氛围空旷、克制、荒凉，有低频风声般的持续底色，少量埙、箫、远鼓、弦鸣质感，但不要明显主旋律。整体像凡人车队穿过不可理解的荒外，不要史诗化，不要仙侠热血，不要现代电子节拍，不要人声歌词，不要模仿任何具体作品。
```

## TRV-001 木轮稳定行进

```text
生成一段 12 秒可无缝循环的古代木轮车队行进音效。包含木轮压过土路、车轴轻微吱呀、绳索和行囊轻晃、远处微风。节奏稳定但不要太机械，不要现代车辆声，不要马车西部片风格，不要音乐旋律，不要人声。
```

## SAN-002 神志重度污染

```text
生成一段 15 秒可循环的低神志认知污染环境声。声音应诡异、克制、遥远：水声从不正确的方向传来，远鼓节拍轻微错位，低语像倒放但不可辨认，木轮声短暂重复。不要尖叫，不要恐怖片跳吓，不要清晰语言，不要歌词，不要强烈噪音墙。
```
