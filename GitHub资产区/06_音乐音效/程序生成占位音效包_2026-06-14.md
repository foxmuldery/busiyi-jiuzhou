## 4. 接入建议

- UI 短音效可以默认随音频开关启用，音量建议 `0.35-0.50`。
- 告警音效只在资源第一次跌破阈值时触发，避免每回合重复打断阅读。
- `SAN-001-SYN` 建议在神志低于 55 时淡入，音量 `0.06-0.14`。
- `SAN-002-SYN` 建议在神志低于 25 时淡入，音量 `0.08-0.22`。
- 两个神志循环层需要 1500ms 以上淡入淡出，页面隐藏时暂停或降到静音。
