## 暂不直接纳入仓库的大类

| 类型 | 处理方式 |
|---|---|
| 未筛选的大图片、视频、音频 | 先登记路径和来源，暂不直接入 Git |
| 原始 ZIP 与解压 staging | 源文件保留在本地 Downloads 或临时区，Git 只提交筛选整理版 |
| 未授权参考图 | 不入库 |
| 竞品截图 | 仅做本地研究索引，不发布 |
| 密钥、token、账号配置 | 不入库 |
| 自动生成的构建产物 | 不入库 |
