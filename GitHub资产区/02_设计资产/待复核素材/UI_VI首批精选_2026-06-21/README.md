## 使用原则

1. 第一轮只做小范围试装，不改整体布局。
2. 优先用于资源状态、路线地图、事件选择区三个高频区域。
3. 小屏先测试 `24px / 32px / 48px` 三档可读性。
4. 面板底板只能轻量叠加，不得压低事件文本对比度。
5. 任何 AI 字样或标题字都不进入正式版本；本目录未纳入文字候选。
