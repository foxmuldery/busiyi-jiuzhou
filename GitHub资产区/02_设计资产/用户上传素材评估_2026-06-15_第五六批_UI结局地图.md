## 5. 风险点

1. AI 生成图仍需 E 线程做授权和发布边界复核。
2. 绿幕自动抠图版是快速处理结果，可能存在细小绿边，正式 UI 切片前需要人工检查。
3. 片名概念图中的“不思异 TV”属于重绘特装方向，不应替代品牌固定台标。
4. 结局/失败图目前没有文字，符合后期叠 UI 的原则；不要再让生成工具直接写中文结局标题。
