# 《不思异：九州》生图提示语工作台

> 用途：给用户在 ChatGPT 网页版生成大图，再由主线程/美术资产流程裁切成 Web Demo 可用的横向旅途舞台素材。

## 1. 通用生成原则

- 优先生成超宽横图，建议比例 `3:1` 或 `4:1`，如工具不支持则选最宽画幅。
- 不要生成文字、Logo、UI、按钮、商标、水印。
- 不要引用任何竞品名称、竞品画风、现代画师姓名或具体插画。
- 背景图不要出现车队、主角、随从或人物主体；车队必须单独生成，后续作为前景行动层叠加。
- 画面要能拆层：天空、远山、中景地貌、前景道路、异象层。车队剪影单独走 `IMG-005`。
- 网页版生图不要要求“透明背景”，模型常会生成黑白棋盘格假透明。需要抠图的素材统一要求纯绿色绿幕或纯色抠图底。
- 绿幕素材要求硬边、清晰轮廓、少虚化、少半透明、无光晕、无投影、无环境反光，主体边缘不要有复杂烟雾或发光渐变。
- 视觉关键词：古代九州、山海图、帛书、矿物色、墨线、朱砂、雾气、荒外远行、神秘但克制。
- 避免：泛仙侠、网游海报、过度华丽人物、赛博蓝紫、怪物图鉴、血腥恐怖、现代建筑。

## 11. 第二轮生成目标

第二轮不再只追求一张漂亮主视觉，而是补齐 Web Demo 需要的“横向假动态舞台素材”。

建议分两条生图线程：

- 线程 1：优先跑 A 组。目标是拿到 3 段可横向移动的背景长图。
- 线程 2：优先跑 B 组和 C 组。目标是拿到可裁切、可叠加、可做动效的素材层。

重要规则：

- A 组、B 组背景素材都不要出现车队、人物主体、随从、动物坐骑。
- C1 是唯一允许出现车队的提示语，必须作为单独前景行动层。
- 所有图都不要文字、Logo、UI、水印。
- C 组不要求透明背景，统一优先用纯绿色绿幕或纯色抠图底。不要生成黑白棋盘格假透明。
- 需要抠图的主体边缘必须清晰，不要柔边、光晕、投影、运动模糊、半透明烟雾或复杂渐变。
