# 《不思异：九州》生图提示语工作台

> 用途：给用户在 ChatGPT 网页版生成大图，再由主线程/美术资产流程裁切成 Web Demo 可用的横向旅途舞台素材。

## 1. 通用生成原则

- 优先生成超宽横图，建议比例 `3:1` 或 `4:1`，如工具不支持则选最宽画幅。
- 不要生成文字、Logo、UI、按钮、商标、水印。
- 不要引用任何竞品名称、竞品画风、现代画师姓名或具体插画。
- 画面要能拆层：天空、远山、中景地貌、前景道路、车队剪影、异象层。
- 视觉关键词：古代九州、山海图、帛书、矿物色、墨线、朱砂、雾气、荒外远行、神秘但克制。
- 避免：泛仙侠、网游海报、过度华丽人物、赛博蓝紫、怪物图鉴、血腥恐怖、现代建筑。

## 3. IMG-001 横向山海旅途主视觉

### 中文提示语

```text
请生成一张超宽横向游戏场景概念图，比例 4:1，主题是中国古代神秘九州远行。

画面像一卷正在展开的山海图：一支很小的古代车队或辇车沿着荒外道路从左向右行进，远处有层叠山影、云气、河渡、山隘、废弃祭所和隐约的墟市轮廓。整体是古籍帛书与矿物颜料质感，墨线克制，局部有朱砂标记。气氛神秘、辽阔、危险但不血腥。

构图要求：画面横向延展，前景、中景、远景层次清楚，适合后期裁切成横向卷轴舞台。车队只占画面很小比例，不要做成主角肖像。不要文字，不要 Logo，不要 UI，不要现代建筑，不要科幻元素，不要明显仙侠网游风。
```

### English prompt

```text
Create an ultra-wide 4:1 panoramic game environment concept art for an ancient mysterious journey across a mythic Chinese landscape.

The image should feel like an unfolding hand-painted Shan Hai scroll: a tiny ancient caravan or ceremonial carriage travels from left to right along a remote road. In the distance are layered mountain silhouettes, drifting mist, a river crossing, a mountain pass, an abandoned ritual altar, and the faint outline of a strange frontier market. Use an aged silk manuscript texture, restrained ink lines, mineral pigments, and subtle cinnabar accents. The mood is mysterious, vast, dangerous, but not gory.

Composition: strong horizontal travel direction, clearly separable foreground, midground, and background layers, suitable for later slicing into a side-scrolling travel stage. The caravan should be small, not a portrait focus. No text, no logo, no UI, no modern buildings, no sci-fi elements, no generic fantasy MMO style.
```

## 4. IMG-002 远景山影/天空层

```text
生成一张超宽横向背景层，比例 4:1，只表现远景天空、云气、山影和日月光色，不要人物，不要车队，不要前景道路。

主题是中国古代神秘九州边境，像古籍山海图中的远山与天象。远山层叠，云缓慢流动的感觉，天空从黄昏过渡到深青夜色，可见淡淡星象和不确定的异象轮廓。画面要干净，方便作为游戏后景层裁切和横向滚动。

风格：帛书纹理、矿物色、淡墨山影、克制朱砂。不要文字、Logo、UI、现代建筑、强人物、过度恐怖。
```

## 5. IMG-003 中景道路/地貌层

```text
生成一张超宽横向中景地貌层，比例 4:1，表现古代九州荒外道路。

画面包含蜿蜒土路、浅水河渡、碎石山隘、低矮祭坛、远处墟市轮廓和一些不可完全辨认的古老地标。不要天空占太多，不要人物肖像，不要 UI。道路从左到右延伸，适合表现车队行进。

风格是古籍山海图与手绘游戏场景结合，墨线清晰但不过度漫画化，矿物色低饱和，局部朱砂符号可出现在石碑或路标上，但不要可读文字。
```

## 6. IMG-004 前景草木/车辙层

```text
生成一张超宽横向前景素材层，比例 4:1，只表现近处路面、草木、石块、车辙、破碎绳索、枯枝和少量风沙。

这是游戏横向旅途舞台的前景视差层，需要底部元素丰富、上方尽量留空，方便叠加在中景道路前面。不要人物，不要完整建筑，不要文字，不要 UI。

风格为古代九州荒外旅途，帛书质感、墨线、低饱和矿物色，气氛苍凉、神秘、克制。
```

## 7. IMG-005 车队/辇车剪影

```text
生成一组适合游戏 UI 使用的古代远行车队剪影，透明或纯色背景优先。

主体是一辆古代辇车或简陋车队，带有木轮、车轴、行囊、布幔和少量随从剪影，侧面视角，朝向右方行进。造型应朴素、疲惫、有旅途感，不要豪华宫廷车，不要战车，不要现代马车，不要西部牛仔马车。

风格：墨色剪影、少量朱砂点缀、古籍插图感。要求轮廓清楚，适合缩小成地图上的当前位置图标。不要文字、Logo、UI。
```

## 8. IMG-006 神志污染异象层

```text
生成一张超宽横向透明叠加层风格的异象概念图，比例 4:1，用于表现低神志状态。

画面中应有错位的山影、重复的道路、若隐若现的假路、模糊的兽形剪影、红色墨痕、断裂的星象和像被水浸坏的古地图纹理。它应该像一层可以叠在正常旅途背景上的不可信幻觉，不要做成独立怪物海报。

气氛诡异、克制、认知污染感强，但不要血腥，不要恐怖脸部特写，不要文字、Logo、UI，不要现代元素。
```

## 9. 生成后回传建议

每张图建议按这个格式命名或说明：

```text
IMG-001-A 横向山海旅途主视觉 4:1
IMG-001-B 横向山海旅途主视觉 4:1
IMG-002-A 远景山影天空层
```

回传时最好附一句主观判断：

```text
这张更像古地图 / 这张更像游戏背景 / 这张太仙侠 / 这张适合裁前景
```

## 10. 第一轮验收标准

- 是否有清楚横向行进方向。
- 是否能拆出前景、中景、远景。
- 是否有山海九州气质，而不是泛仙侠。
- 是否没有文字、Logo、UI、水印。
- 是否能作为 Web Demo 的舞台资产，而不是单张插画。
