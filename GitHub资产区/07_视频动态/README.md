# 《不思异：九州》视频动态资产区

这个目录用于管理 Web Demo 和后续版本的视频动态素材，包括短循环背景、路线转场、神志异象、关键地点显影、结局镜头、视频生成提示语和授权记录。

## 目录结构

| 目录 | 用途 |
|---|---|
| `00_视频台账` | 视频资产编号、来源、授权、用途、体积和状态 |
| `01_旅途转场` | 横向旅途、日夜变化、路线移动、地点显影方向 |
| `02_神志异象` | 低神志污染、重影、幻路、第二月亮、异常云气方向 |
| `03_生成提示语` | AI 视频生成提示语、参数、负面约束与工具记录 |
| `04_待复核素材` | 已生成但未验收、需压缩或需复核授权的视频 |
| `05_可用素材` | 已验收、可进入 Web Demo 的视频素材 |

## 使用原则

1. 视频是增强层，不替代节点路线、资源判断和事件阅读。
2. Web Demo 第一版优先 3 到 8 秒短循环，不做长剧情动画。
3. 视频默认静音，音乐和音效由 F 线程单独控制。
4. 每个视频必须记录来源、工具、日期、提示语、用途、授权状态和成本。
5. 大视频文件先放待复核区或本地外部素材盘，未经主线程批准不直接进 Git。
6. 需要抠像的前景元素用绿幕或纯色背景，不要求透明图层。
7. 避免黑白棋盘格背景、虚化光晕边缘、过度软边和不可控烟雾。
8. 背景视频原则上不带车队，车队应作为前景独立层，方便复用和遮挡控制。

## 第一版推荐用途

| 用途 | 推荐长度 | 是否循环 | 默认优先级 |
|---|---:|---|---|
| 横向旅途云影/雾气 | 4-8 秒 | 是 | 高 |
| 日夜色调变化 | 3-6 秒 | 否或短循环 | 中 |
| 神志轻度异象 | 4-8 秒 | 是 | 高 |
| 神志重度异象 | 4-8 秒 | 是 | 中 |
| 关键地点显影 | 3-6 秒 | 否 | 中 |
| 结局短镜头 | 8-15 秒 | 否 | 低 |

## Web 技术建议

- 优先准备 `.webm`，必要时补 `.mp4`。
- 每条视频准备一张同构静态 poster 图。
- 第一版单条视频尽量控制在 1 到 4 MB 内。
- 移动端、低性能设备或省流模式下回退为静态图。
- 循环视频的首尾光照、云气、雾气和运动方向要能无缝衔接。
