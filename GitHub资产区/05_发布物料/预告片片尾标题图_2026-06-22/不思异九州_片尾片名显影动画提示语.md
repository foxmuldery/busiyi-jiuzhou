## 推荐剪辑层级

最底层：无字地图首帧，持续 5-6 秒，可做 102%-104% 极慢推近。

背景氛围层：`VID-OV-CLOUD-001`，混合模式建议 Screen / Lighten，透明度 12%-22%，速度 70%-90%，主要放在画面上半部和边缘，不能盖住最后的片名。

地图显影层：`VID-OV-MAPREVEAL-001`，混合模式建议 Screen / Overlay，透明度 8%-16%，0.6-2.4 秒出现，让无字地图有旧帛被雾擦开的感觉。

裂隙预兆层：`VID-OV-CRACK-001`，混合模式建议 Screen / Add，透明度 10%-20%，1.0-2.6 秒短暂出现，放在标题中心后方，不能压过九州笔画。

印章显现层：`VID-OV-SEAL-001`，混合模式建议 Screen / Add，透明度 18%-32%，1.8-3.0 秒只作用在“不思异TV”红印附近。红印本体仍以尾帧或透明片名为准，不让视频重新写字。

金尘题字层：`VID-OV-GOLD-001`，混合模式建议 Screen / Add，透明度 12%-28%，2.6-4.1 秒从下方或中心扫过，用来带出“九州”。

纸尘收束层：`VID-OV-DUST-001` 或 `VID-OV-SAND-001`，混合模式建议 Screen / Lighten，透明度 8%-18%，3.8-5.6 秒轻轻收尾。风沙不要太强，片尾最后 1 秒要能看清字。

最终定格层：尾帧标题图，4.5 秒后完全落定，最后至少停留 1 秒。尾帧上方可保留极淡云烟，但不要再有明显裂纹脉冲或粒子爆闪。
